# HTML `<!-- ... -->` Comment

The HTML comment syntax `<!-- ... -->` is used to add comments to the source code.

Comments are **not displayed** in the browser. They are only visible in the HTML source code and are ignored by the browser when rendering the page.

Comments are useful for:
- Explaining sections of your code.
- Leaving notes or reminders for yourself or other developers.
- Temporarily disabling parts of the code during testing or debugging.

## Syntax

```html
<!-- This is a comment -->
```

## Example

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>Title of the Document</title>
  </head>
  <body>
    <h1>This is a heading</h1>

    <!-- This is a comment. Comments are not displayed in the browser. -->

    <p>This is a paragraph.</p>
  </body>
</html>
```
