# HTML `<title>` Tag

The `<title>` tag defines the **title of an HTML document**. The title must contain **text only** and is displayed in the browser's **tab**, **title bar**, and **search engine results**.

The `<title>` element is **required** in every HTML document and must be placed inside the `<head>` section.

## Syntax

```html
<head>
  <title>Page Title</title>
</head>
```

## Example

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <title>HTML Elements Reference</title>
</head>
<body>
  <h1>This is a heading</h1>
  <p>This is a paragraph.</p>
</body>
</html>
```

## Why Is the `<title>` Tag Important?

The `<title>` element serves several important purposes:

- Displays the page title in the browser tab or title bar.
- Provides the title when the page is bookmarked or added to favorites.
- Appears as the clickable headline in search engine results.
- Helps search engines understand the content of the page, making it important for **Search Engine Optimization (SEO)**.
- Improves the user experience by helping visitors identify the page.

## Best Practices

- Write a unique title for every page.
- Keep the title clear, descriptive, and relevant to the page content.
- Place the most important keywords near the beginning of the title.
- Keep the title reasonably short (typically **50–60 characters**) so it is less likely to be truncated in search results.
