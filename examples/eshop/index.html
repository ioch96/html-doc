Σχετικά με εμάς

Το TechStore είναι ένα εκπαιδευτικό ηλεκτρονικό κατάστημα που δημιουργήθηκε ως προσωπικό project για portfolio.

Στόχος του project είναι να επεκταθεί σταδιακά με CSS, JavaScript και τεχνολογίες backend.

Το project δίνει έμφαση στη σωστή σημασιολογική δομή της HTML, στην προσβασιμότητα και στη δημιουργία εύχρηστων φορμών.
