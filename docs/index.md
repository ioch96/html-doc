# Το αρχείο `index.html`

Το **`index.html`** αποτελεί το βασικό και σημαντικότερο αρχείο μιας ιστοσελίδας, καθώς λειτουργεί ως η **προεπιλεγμένη αρχική σελίδα (homepage)** ενός ιστότοπου. Όταν ένας χρήστης επισκέπτεται μια διεύθυνση URL χωρίς να καθορίζει κάποιο συγκεκριμένο αρχείο, ο web server αναζητά αυτόματα το αρχείο `index.html` και το εμφανίζει στον φυλλομετρητή (browser).

Για παράδειγμα, όταν επισκεπτόμαστε τη διεύθυνση:

```text
https://www.example.com/
```

ο web server θα εμφανίσει το αρχείο:

```text
index.html
```

χωρίς να είναι απαραίτητο να πληκτρολογηθεί η πλήρης διαδρομή:

```text
https://www.example.com/index.html
```

## Ρόλος του `index.html`

Το αρχείο `index.html` αποτελεί το σημείο εισόδου (entry point) ενός website και περιέχει τη βασική δομή της ιστοσελίδας. Μέσα σε αυτό μπορούν να ενσωματωθούν:

- Επικεφαλίδες και παράγραφοι κειμένου.
- Εικόνες και πολυμέσα.
- Υπερσύνδεσμοι (links) προς άλλες σελίδες.
- Πίνακες και λίστες.
- Φόρμες εισαγωγής δεδομένων.
- Αναφορές σε αρχεία CSS για τη μορφοποίηση της σελίδας.
- Αναφορές σε αρχεία JavaScript για την προσθήκη διαδραστικότητας.

Το `index.html` είναι συνήθως η πρώτη σελίδα που βλέπει ο επισκέπτης και λειτουργεί ως το κεντρικό σημείο πλοήγησης προς τις υπόλοιπες σελίδες του ιστότοπου.

## Η βασική δομή ενός αρχείου `index.html`

Κάθε αρχείο HTML ακολουθεί μια συγκεκριμένη δομή. Ένα απλό παράδειγμα είναι το παρακάτω:

```html
<!DOCTYPE html>
<html lang="el">
<head>
    <meta charset="UTF-8">
    <title>Η πρώτη μου ιστοσελίδα</title>
</head>

<body>
    <h1>Καλώς ήρθατε!</h1>
    <p>Αυτή είναι η αρχική σελίδα του ιστότοπού μου.</p>
</body>
</html>
```

Στο παραπάνω παράδειγμα:

- Η δήλωση `<!DOCTYPE html>` ενημερώνει τον browser ότι το έγγραφο είναι γραμμένο σε HTML5.
- Το στοιχείο `<html>` περιέχει ολόκληρο το περιεχόμενο της ιστοσελίδας.
- Το `<head>` περιλαμβάνει πληροφορίες για τη σελίδα, όπως τον τίτλο, την κωδικοποίηση χαρακτήρων και συνδέσεις με εξωτερικά αρχεία.
- Το `<body>` περιέχει όλα τα στοιχεία που εμφανίζονται στον χρήστη.

## Σύνδεση με αρχεία CSS και JavaScript

Το `index.html` συνεργάζεται συνήθως με αρχεία CSS και JavaScript.

Παράδειγμα:

```html
<!DOCTYPE html>
<html lang="el">
<head>
    <meta charset="UTF-8">
    <title>Η ιστοσελίδα μου</title>

    <link rel="stylesheet" href="style.css">
</head>

<body>

    <h1>Καλώς ήρθατε!</h1>

    <script src="script.js"></script>

</body>
</html>
```

Στο παράδειγμα αυτό:

- Το αρχείο `style.css` καθορίζει την εμφάνιση της ιστοσελίδας.
- Το αρχείο `script.js` προσθέτει λειτουργίες και διαδραστικότητα.

## Παράδειγμα δομής ενός website

Ένα απλό website μπορεί να έχει την ακόλουθη δομή:

```text
website/
│
├── index.html
├── about.html
├── contact.html
├── style.css
├── script.js
└── images/
    ├── logo.png
    └── banner.jpg
```

Στην παραπάνω δομή:

- Το `index.html` είναι η αρχική σελίδα.
- Το `about.html` περιέχει πληροφορίες για τον ιστότοπο.
- Το `contact.html` περιλαμβάνει στοιχεία επικοινωνίας.
- Το `style.css` ορίζει τη μορφοποίηση.
- Το `script.js` περιέχει τον κώδικα JavaScript.
- Ο φάκελος `images` αποθηκεύει τις εικόνες του website.

## Είναι υποχρεωτικό το όνομα `index.html`

Η HTML δεν απαιτεί το αρχείο να ονομάζεται υποχρεωτικά `index.html`. Μπορεί να χρησιμοποιηθεί οποιοδήποτε όνομα, όπως `home.html` ή `main.html`.

Ωστόσο, οι περισσότεροι web servers έχουν ως προεπιλογή να αναζητούν το αρχείο `index.html` όταν ένας χρήστης επισκέπτεται έναν φάκελο ή την αρχική διεύθυνση ενός website. Για τον λόγο αυτό αποτελεί τη διεθνώς καθιερωμένη πρακτική στην ανάπτυξη ιστοσελίδων.

## Συμπέρασμα

Το **`index.html`** είναι το σημαντικότερο αρχείο ενός website, καθώς αποτελεί την αρχική σελίδα που φορτώνεται αυτόματα από τον web server. Περιέχει τη βασική δομή της ιστοσελίδας, συνδέεται με αρχεία CSS και JavaScript και λειτουργεί ως το κεντρικό σημείο πλοήγησης προς το υπόλοιπο περιεχόμενο του ιστότοπου. Για τον λόγο αυτό, αποτελεί το πρώτο αρχείο που δημιουργείται κατά την ανάπτυξη μιας νέας ιστοσελίδας.
