# Emoji στην HTML

## Τι είναι τα Emoji

Τα emoji μοιάζουν με εικόνες, αλλά στην πραγματικότητα δεν είναι.

Τα emoji είναι εικονογράμματα (σύμβολα) που εμφανίζονται με έγχρωμη μορφή και χρησιμοποιούνται μέσα στο κείμενο. Αναπαριστούν πρόσωπα, καιρικά φαινόμενα, οχήματα, κτίρια, φαγητά και ποτά, ζώα, φυτά, καθώς και σύμβολα που εκφράζουν συναισθήματα, δραστηριότητες ή διάφορες έννοιες.

Τα emoji είναι χαρακτήρες του συνόλου χαρακτήρων **UTF-8 (Unicode)**:

😄 😍 💗

Η κωδικοποίηση **UTF-8** περιλαμβάνει σχεδόν όλους τους χαρακτήρες και τα σύμβολα που χρησιμοποιούνται σε όλο τον κόσμο.

## Χρήση Emoji στην HTML

Τα emoji μπορούν να χρησιμοποιηθούν στην HTML όπως οποιοσδήποτε άλλος χαρακτήρας κειμένου. Εφόσον η ιστοσελίδα χρησιμοποιεί την κωδικοποίηση **UTF-8**, τα emoji θα εμφανίζονται σωστά στους περισσότερους σύγχρονους φυλλομετρητές (browsers).

Για να εξασφαλίσετε ότι η σελίδα σας χρησιμοποιεί UTF-8, προσθέστε την παρακάτω ετικέτα μέσα στο στοιχείο `<head>`:

```html
<meta charset="UTF-8">
```

### Εισαγωγή Emoji απευθείας

Ο πιο απλός τρόπος είναι να εισάγετε το emoji απευθείας μέσα στον HTML κώδικα.

```html
<p>😀 Καλώς ήρθατε!</p>
<p>😎 Μαθαίνουμε HTML.</p>
<p>🎉 Συγχαρητήρια!</p>
```

### Χρήση του δεκαεξαδικού κωδικού (Hex)

Κάθε emoji διαθέτει έναν μοναδικό δεκαεξαδικό κωδικό Unicode. Για να εμφανίσετε ένα emoji, χρησιμοποιήστε τη σύνταξη:

```html
&#xΚΩΔΙΚΟΣ;
```

Παράδειγμα:

```html
<p>&#x1F600;</p>
<p>&#x1F44D;</p>
<p>&#x1F389;</p>
```

**Αποτέλεσμα:**

😀 👍 🎉

### Χρήση του δεκαδικού κωδικού (Decimal)

Μπορείτε επίσης να χρησιμοποιήσετε τον δεκαδικό αριθμό του emoji.

Παράδειγμα:

```html
<p>&#128512;</p>
<p>&#128077;</p>
<p>&#127881;</p>
```

**Αποτέλεσμα:**

😀 👍 🎉

> **Σημείωση:** Στους παρακάτω πίνακες εμφανίζονται οι πιο συνηθισμένοι χαρακτήρες emoji μαζί με τους δεκαδικούς (**Dec**) και δεκαεξαδικούς (**Hex**) κωδικούς τους, ώστε να μπορείτε να τους χρησιμοποιείτε στην HTML.
