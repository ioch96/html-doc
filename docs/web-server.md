# Τι είναι ένας Web Server

Ένας **web server** είναι ένα πρόγραμμα που αποθηκεύει τα αρχεία μιας ιστοσελίδας και τα στέλνει στους browsers όταν αυτά τα ζητούν.

Για παράδειγμα, όταν επισκέπτεσαι μια ιστοσελίδα, ο browser επικοινωνεί με τον web server που τη φιλοξενεί. Ο web server επιστρέφει τα αρχεία της ιστοσελίδας, όπως το HTML, τη CSS, τη JavaScript και τις εικόνες.

Στις περισσότερες περιπτώσεις δεν χρειάζεται να δημιουργήσεις εσύ έναν web server. Αν ανεβάσεις την ιστοσελίδα σου σε μια υπηρεσία φιλοξενίας (hosting), ο web server υπάρχει ήδη και αναλαμβάνει να εξυπηρετεί τους επισκέπτες.
