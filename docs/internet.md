# Το Internet

Το **Internet (Διαδίκτυο)** είναι ένα παγκόσμιο δίκτυο που συνδέει εκατομμύρια υπολογιστές και άλλες συσκευές μεταξύ τους.

Χάρη στο Internet, δύο υπολογιστές που βρίσκονται σε διαφορετικά σημεία του κόσμου μπορούν να ανταλλάσσουν δεδομένα και να επικοινωνούν μεταξύ τους.

Με απλά λόγια, το Internet είναι η υποδομή που επιτρέπει τη μεταφορά πληροφοριών από έναν υπολογιστή σε έναν άλλον.

---

## Το Internet δεν είναι οι ιστοσελίδες

Πολλοί πιστεύουν ότι το Internet και οι ιστοσελίδες είναι το ίδιο πράγμα.

Στην πραγματικότητα, πρόκειται για δύο διαφορετικές έννοιες.

Το **Internet** είναι το παγκόσμιο δίκτυο που επιτρέπει την επικοινωνία μεταξύ υπολογιστών και τη μεταφορά δεδομένων.

Για παράδειγμα, μέσω του Internet μπορούμε να:

* επισκεπτόμαστε ιστοσελίδες,
* στέλνουμε και να λαμβάνουμε email,
* πραγματοποιούμε βιντεοκλήσεις,
* παρακολουθούμε βίντεο και μουσική μέσω υπηρεσιών streaming,
* παίζουμε online παιχνίδια,
* ανταλλάσσουμε μηνύματα μέσω εφαρμογών επικοινωνίας.

Όλες αυτές οι δραστηριότητες χρησιμοποιούν το ίδιο δίκτυο, δηλαδή το Internet.

Με άλλα λόγια, το Internet είναι η υποδομή που μεταφέρει τα δεδομένα, ενώ οι ιστοσελίδες αποτελούν μία από τις εφαρμογές που αξιοποιούν αυτή την υποδομή.

---

## Πώς χρησιμοποιείται το Internet

Όταν επισκέπτεσαι μια ιστοσελίδα, ο browser επικοινωνεί με έναν απομακρυσμένο υπολογιστή που φιλοξενεί την ιστοσελίδα.

Το Internet μεταφέρει τα δεδομένα ανάμεσα στον browser και στον υπολογιστή αυτόν.

Η διαδικασία μπορεί να απεικονιστεί ως εξής:

```text
Browser
    │
    │ Αίτημα
    ▼
════════ Internet ════════
    ▲
    │ Απάντηση
Web Server
```

Ο browser στέλνει ένα αίτημα μέσω του Internet και ο Web Server επιστρέφει τα αρχεία της ιστοσελίδας μέσω του ίδιου δικτύου.

---

## Τι μεταφέρεται μέσω του Internet

Ο Web Server στέλνει στον browser όλα τα απαραίτητα αρχεία της ιστοσελίδας, όπως:

* HTML,
* CSS,
* JavaScript,
* εικόνες,
* γραμματοσειρές,
* βίντεο,
* και κάθε άλλο απαραίτητο αρχείο.

Ο browser λαμβάνει αυτά τα αρχεία και τα χρησιμοποιεί για να δημιουργήσει την ιστοσελίδα που βλέπει ο χρήστης.

---

## Σύνοψη

Το Internet είναι το παγκόσμιο δίκτυο που επιτρέπει την επικοινωνία μεταξύ υπολογιστών.

Δεν αποτελείται από ιστοσελίδες, αλλά χρησιμοποιείται για τη μεταφορά των δεδομένων τους.

Χωρίς το Internet, ο browser δεν θα μπορούσε να επικοινωνήσει με τον Web Server και να λάβει τα αρχεία μιας ιστοσελίδας.
