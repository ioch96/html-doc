# Προσθήκη Favicon

Το **favicon** (favorite icon) είναι το μικρό εικονίδιο που εμφανίζεται στην καρτέλα (tab) του browser, στα αγαπημένα (Bookmarks) και σε άλλες περιοχές του προγράμματος περιήγησης. Χρησιμοποιείται για την αναγνώριση μιας ιστοσελίδας και αποτελεί μέρος της εταιρικής ταυτότητας (branding) ενός website.

Το favicon δηλώνεται μέσα στο στοιχείο `<head>` με τη χρήση του tag `<link>`.

## Σύνταξη

```html
<head>
    <link rel="icon" href="favicon.ico">
</head>
```

Εάν το αρχείο βρίσκεται σε διαφορετικό φάκελο, καθορίζουμε την αντίστοιχη διαδρομή (path).

Παράδειγμα:

```html
<head>
    <link rel="icon" href="images/favicon.png">
</head>
```

> **Σημείωση:** Το attribute `type` είναι προαιρετικό και χρησιμοποιείται για να δηλώσει τον τύπο (MIME type) του favicon, όπως `image/png`. Οι περισσότεροι σύγχρονοι browsers αναγνωρίζουν αυτόματα τον τύπο του αρχείου, επομένως συνήθως δεν είναι απαραίτητο να οριστεί.

```html
<head>
    <link rel="icon" href="favicon.png" type="image/png">
</head>
```

## Υποστηριζόμενες μορφές

Τα πιο συνηθισμένα αρχεία favicon είναι:

- `.ico`
- `.png`
- `.svg`
- `.jpg` ή `.jpeg` (λιγότερο συνηθισμένα)

> **Σημείωση:** Η μορφή **`.ico`** προσφέρει τη μεγαλύτερη συμβατότητα με όλους τους browsers, ενώ η μορφή **`.png`** χρησιμοποιείται ευρέως στις σύγχρονες ιστοσελίδες.

## Πού εμφανίζεται το favicon

Το favicon εμφανίζεται:

- Στην καρτέλα (tab) του browser.
- Στη λίστα των αγαπημένων (Bookmarks).
- Στο ιστορικό περιήγησης (History).
- Σε συντομεύσεις της ιστοσελίδας που αποθηκεύονται στην επιφάνεια εργασίας ή στην αρχική οθόνη κινητών συσκευών.

## Best Practices

- Χρησιμοποιείτε τετράγωνες εικόνες (π.χ. **32×32**, **48×48** ή **64×64** pixels).
- Προτιμήστε μορφή **PNG** ή **ICO** για μέγιστη συμβατότητα.
- Τοποθετήστε το favicon μέσα στο στοιχείο `<head>`.
- Χρησιμοποιήστε ένα απλό και ευδιάκριτο εικονίδιο, ώστε να αναγνωρίζεται εύκολα ακόμη και σε μικρό μέγεθος.
