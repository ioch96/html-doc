# Emoji στην HTML

## Τι είναι τα Emoji

Τα emoji μοιάζουν με εικόνες, αλλά στην πραγματικότητα είναι χαρακτήρες του προτύπου **Unicode**.

Πρόκειται για εικονογράμματα (σύμβολα) που εμφανίζονται έγχρωμα μέσα στο κείμενο και χρησιμοποιούνται για να αναπαραστήσουν πρόσωπα, συναισθήματα, καιρικά φαινόμενα, ζώα, φυτά, οχήματα, κτίρια, φαγητά, ποτά, δραστηριότητες και πολλές ακόμη έννοιες.

Τα emoji αποτελούν μέρος του συνόλου χαρακτήρων **UTF-8 (Unicode)**.

Η κωδικοποίηση **UTF-8** περιλαμβάνει σχεδόν όλους τους χαρακτήρες και τα σύμβολα που χρησιμοποιούνται στις περισσότερες γλώσσες του κόσμου.

## Χρήση Emoji στην HTML

Τα emoji μπορούν να χρησιμοποιηθούν στην HTML όπως οποιοσδήποτε άλλος χαρακτήρας κειμένου. Αρκεί η ιστοσελίδα να χρησιμοποιεί την κωδικοποίηση **UTF-8**, ώστε τα emoji να εμφανίζονται σωστά στους περισσότερους σύγχρονους φυλλομετρητές (browsers).

Για να εξασφαλίσετε ότι η σελίδα σας χρησιμοποιεί UTF-8, προσθέστε την παρακάτω ετικέτα μέσα στο στοιχείο `<head>`:

```html
<meta charset="UTF-8">
```

## Τρόποι εισαγωγής Emoji

Υπάρχουν τρεις τρόποι για να εισαγάγετε ένα emoji στην HTML:

* **Απευθείας ως χαρακτήρα** (προτεινόμενος τρόπος)
* **Με τον δεκαεξαδικό κωδικό Unicode (Hex)**
* **Με τον δεκαδικό κωδικό Unicode (Decimal)**

### Απευθείας εισαγωγή

Ο πιο απλός και σύγχρονος τρόπος είναι να εισάγετε το emoji απευθείας μέσα στον HTML κώδικα.

Παράδειγμα:

```html
<p>😀 Καλώς ήρθατε!</p>
<p>😎 Μαθαίνουμε HTML.</p>
<p>🎉 Συγχαρητήρια!</p>
```

Πλεονεκτήματα:

* Είναι ο πιο απλός και ευανάγνωστος τρόπος.
* Υποστηρίζεται από όλους τους σύγχρονους browsers.
* Προτείνεται για τις περισσότερες ιστοσελίδες.

### Χρήση δεκαεξαδικού κωδικού (Hex)

Κάθε emoji διαθέτει έναν μοναδικό δεκαεξαδικό κωδικό Unicode. Για να εμφανίσετε ένα emoji, χρησιμοποιήστε τη σύνταξη:

```html
&#xΚΩΔΙΚΟΣ;
```

Παράδειγμα:

```html
<p>&#x1F600;</p>
<p>&#x1F44D;</p>
<p>&#x1F389;</p>
```

Αποτέλεσμα:

```none
😀 👍 🎉
```

* Χρησιμοποιεί τον δεκαεξαδικό κωδικό Unicode.
* Θα τον συναντήσετε συχνά στην τεκμηρίωση του Unicode και σε πίνακες emoji.

### Χρήση δεκαδικού κωδικού (Decimal)

Μπορείτε επίσης να χρησιμοποιήσετε τον δεκαδικό αριθμό του emoji.

Παράδειγμα:

```html
<p>&#128512;</p>
<p>&#128077;</p>
<p>&#127881;</p>
```

Αποτέλεσμα:

```none
😀 👍 🎉
```

* Είναι ισοδύναμος με τον Hex κωδικό.
* Χρησιμοποιείται κυρίως σε παλαιότερα παραδείγματα ή όταν οι κωδικοί Unicode δίνονται σε δεκαδική μορφή.

> Στις σύγχρονες ιστοσελίδες προτείνεται η απευθείας εισαγωγή του emoji στον HTML κώδικα, καθώς είναι πιο ευανάγνωστη και δεν διαφέρει λειτουργικά από τη χρήση των δεκαδικών (Decimal) ή δεκαεξαδικών (Hex) κωδικών Unicode. Οι κωδικοί αυτοί χρησιμοποιούνται κυρίως για λόγους συμβατότητας, σε παλαιότερο κώδικα ή όταν παρέχονται από πίνακες Unicode.
