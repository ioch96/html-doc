# Ο World Wide Web (WWW)

Ο **World Wide Web (WWW)** ή **Παγκόσμιος Ιστός** είναι μία υπηρεσία που λειτουργεί πάνω στο Internet.

Αποτελείται από τις ιστοσελίδες και τις web εφαρμογές που είναι διαθέσιμες μέσω του διαδικτύου.

Με απλά λόγια, ο World Wide Web είναι το σύνολο όλων των ιστοσελίδων που μπορούμε να επισκεφθούμε χρησιμοποιώντας έναν browser.

---

## Internet και World Wide Web

Οι όροι **Internet** και **World Wide Web** χρησιμοποιούνται συχνά σαν να σημαίνουν το ίδιο πράγμα.

Στην πραγματικότητα όμως είναι διαφορετικές έννοιες.

Το Internet είναι το παγκόσμιο δίκτυο επικοινωνίας.

Ο World Wide Web είναι μία υπηρεσία που χρησιμοποιεί αυτό το δίκτυο.

Μπορούμε να το φανταστούμε ως εξής:

```text
Internet
│
├── Email
├── Online Games
├── Streaming
└── World Wide Web
     ├── example.com
     ├── wikipedia.org
     ├── github.com
     └── η δική σου ιστοσελίδα
```

---

## Πού βρίσκεται η ιστοσελίδα μου

Η ιστοσελίδα σου αποτελεί μέρος του World Wide Web.

Τα αρχεία της όμως δεν βρίσκονται «μέσα στο Internet».

Αποθηκεύονται σε έναν **Web Server**, δηλαδή σε έναν υπολογιστή που είναι συνδεδεμένος στο Internet.

Όταν κάποιος επισκέπτεται την ιστοσελίδα σου, ο browser επικοινωνεί με αυτόν τον Web Server και ζητά τα αρχεία της.

---

## Πώς λειτουργεί ο World Wide Web

Όταν ένας χρήστης πληκτρολογήσει μια διεύθυνση, όπως:

```text
https://example.com
```

συμβαίνουν τα εξής:

1. Ο browser στέλνει αίτημα μέσω του Internet.
2. Το αίτημα φτάνει στον Web Server.
3. Ο Web Server επιστρέφει τα αρχεία της ιστοσελίδας.
4. Ο browser τα επεξεργάζεται.
5. Η ιστοσελίδα εμφανίζεται στην οθόνη.

Η διαδικασία μπορεί να απεικονιστεί ως εξής:

```text
Χρήστης
    │
    ▼
Browser
    │
    ▼
Internet
    │
    ▼
Web Server
    │
    ▼
HTML, CSS, JavaScript
    │
    ▼
Internet
    │
    ▼
Browser
    │
    ▼
Ιστοσελίδα
```

---

## Γιατί ονομάζεται "Ιστός"

Ονομάζεται **World Wide Web** επειδή οι ιστοσελίδες συνδέονται μεταξύ τους μέσω υπερσυνδέσμων (Hyperlinks).

Ένας σύνδεσμος μπορεί να οδηγεί από μια ιστοσελίδα σε μια άλλη, δημιουργώντας ένα τεράστιο δίκτυο διασυνδεδεμένων σελίδων, παρόμοιο με έναν ιστό.

Για παράδειγμα, μια ιστοσελίδα μπορεί να περιέχει συνδέσμους προς:

* άλλες σελίδες του ίδιου ιστότοπου,
* διαφορετικούς ιστότοπους,
* έγγραφα,
* εικόνες,
* βίντεο.

Οι εκατομμύρια αυτοί σύνδεσμοι είναι που σχηματίζουν τον Παγκόσμιο Ιστό.

---

## Σύνοψη

Ο World Wide Web είναι η υπηρεσία που περιλαμβάνει όλες τις ιστοσελίδες και τις web εφαρμογές του διαδικτύου.

Οι ιστοσελίδες αποθηκεύονται σε Web Servers, μεταφέρονται μέσω του Internet και προβάλλονται από τους browsers.

Στο επόμενο κεφάλαιο θα γνωρίσουμε τον **Web Server**, δηλαδή το πρόγραμμα που αποθηκεύει και αποστέλλει τα αρχεία μιας ιστοσελίδας στους browsers.
