# HTML `<i>` Tag

Το tag `<i>` χρησιμοποιείται για να ορίσει ένα τμήμα κειμένου που εκφράζεται με **διαφορετική φωνή, ύφος ή διάθεση (alternate voice or mood)**.

Το περιεχόμενό του εμφανίζεται συνήθως με **πλάγια γραφή (italic)** στους περισσότερους browsers.

Σε αντίθεση με το `<em>`, το `<i>` **δεν δηλώνει έμφαση**. Χρησιμοποιείται για κείμενο που ξεχωρίζει για σημασιολογικούς λόγους, όπως:

- Τεχνικοί όροι.
- Λέξεις ή φράσεις από άλλη γλώσσα.
- Ονόματα πλοίων.
- Επιστημονικές ονομασίες.
- Σκέψεις ή εσωτερικός μονόλογος.
- Ονόματα έργων (σε ορισμένες περιπτώσεις).

## Παράδειγμα

```html
<!DOCTYPE html>
<html lang="el">
  <head>
    <title>Τίτλος του Εγγράφου</title>
  </head>

  <body>

    <h1>Το στοιχείο i</h1>

    <p>
      Η λέξη <i>cache</i> είναι ένας τεχνικός όρος στην επιστήμη των υπολογιστών.
    </p>

    <p>
      Η γαλλική φράση <i>bon appétit</i> σημαίνει «καλή όρεξη».
    </p>

    <p>
      <i>Μακάρι να είχα περισσότερο χρόνο...</i>
    </p>

  </body>
</html>
```

## Συνήθεις χρήσεις

### Τεχνικοί όροι

```html
<p>
  Η έννοια του <i>polymorphism</i> είναι βασική στην αντικειμενοστραφή
  προγραμματισμό.
</p>
```

### Ξένη λέξη ή φράση

```html
<p>
  Το <i>carpe diem</i> σημαίνει «ζήσε τη στιγμή».
</p>
```

### Σκέψη

```html
<p>
  <i>Πρέπει να βρω μια λύση...</i>
</p>
```

### Όνομα πλοίου

```html
<p>
  Το <i>Titanic</i> βυθίστηκε το 1912.
</p>
```

## Διαφορά μεταξύ `<i>` και `<em>`

|`<i>`|`<em>`|
|--------|---------|
|Δηλώνει διαφορετικό ύφος ή φωνή.|Δηλώνει έμφαση στο κείμενο.|
|Δεν αλλάζει τη σημασία της πρότασης.|Η έμφαση μπορεί να αλλάξει το νόημα της πρότασης.|
|Συνήθως χρησιμοποιείται για τεχνικούς όρους, ξένες λέξεις και σκέψεις.|Χρησιμοποιείται όταν θέλουμε να τονίσουμε μια λέξη ή φράση.|
|Εμφανίζεται συνήθως με πλάγια γραφή.|Επίσης εμφανίζεται συνήθως με πλάγια γραφή.|

## Βέλτιστες Πρακτικές (Best Practices)

- Χρησιμοποιείτε το `<i>` όταν το κείμενο έχει διαφορετικό ύφος ή ειδική σημασιολογική χρήση.
- Μην χρησιμοποιείτε το `<i>` μόνο για να κάνετε το κείμενο πλάγιο. Αν θέλετε μόνο αλλαγή εμφάνισης, χρησιμοποιήστε CSS.
- Αν θέλετε να δώσετε έμφαση στο κείμενο, χρησιμοποιήστε το `<em>` αντί για το `<i>`.

> **Σημείωση:** Παρόλο που τα `<i>` και `<em>` εμφανίζονται συνήθως με πλάγια γραφή, εξυπηρετούν διαφορετικούς σκοπούς. Το `<i>` χρησιμοποιείται για ειδικούς όρους, ξένες λέξεις, σκέψεις ή άλλο ιδιαίτερο ύφος, ενώ το `<em>` χρησιμοποιείται για να δώσει **έμφαση** στο περιεχόμενο.
