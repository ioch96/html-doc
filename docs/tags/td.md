# HTML `<td>` Tag

Το tag `<td>` χρησιμοποιείται για τη δημιουργία ενός **τυπικού κελιού δεδομένων (table data cell)** μέσα σε έναν πίνακα HTML.

Το `<td>` τοποθετείται πάντοτε μέσα σε μια γραμμή (`<tr>`) και περιέχει τα πραγματικά δεδομένα του πίνακα, όπως κείμενο, αριθμούς, εικόνες, συνδέσμους ή άλλα HTML στοιχεία.

---

## Τα δύο είδη κελιών σε έναν πίνακα

Ένας πίνακας HTML αποτελείται από δύο βασικά είδη κελιών:

- **Κελιά επικεφαλίδων (Header cells)**, τα οποία δημιουργούνται με το tag `<th>`.
- **Κελιά δεδομένων (Data cells)**, τα οποία δημιουργούνται με το tag `<td>`.

Συνήθως, τα `<th>` χρησιμοποιούνται στην πρώτη γραμμή ή στην πρώτη στήλη για την περιγραφή των δεδομένων, ενώ τα `<td>` περιέχουν τα πραγματικά δεδομένα του πίνακα.

---

## Παράδειγμα

```html
<table>
    <tr>
        <th>Όνομα</th>
        <th>Ηλικία</th>
    </tr>
    <tr>
        <td>Μαρία</td>
        <td>25</td>
    </tr>
    <tr>
        <td>Γιώργος</td>
        <td>31</td>
    </tr>
</table>
```

---

## Προεπιλεγμένη εμφάνιση

Από προεπιλογή, οι περισσότεροι browsers εμφανίζουν το περιεχόμενο ενός `<td>`:

- με κανονικό (όχι έντονο) κείμενο,
- στοιχισμένο αριστερά.

Αντίθετα, τα στοιχεία `<th>` εμφανίζονται συνήθως:

- με έντονη γραφή (bold),
- κεντραρισμένα.

> Η εμφάνιση αυτή είναι η προεπιλεγμένη συμπεριφορά του browser και μπορεί να αλλάξει με CSS.

---

## Attributes

|Attribute|Value|Περιγραφή|
|:-:|:-:|:-|
|`colspan`|number|Καθορίζει πόσες στήλες θα καταλαμβάνει το κελί.|
|`rowspan`|number|Καθορίζει πόσες γραμμές θα καταλαμβάνει το κελί.|

---

### Το attribute `colspan`

Το `colspan` επιτρέπει σε ένα κελί να εκτείνεται σε περισσότερες από μία στήλες.

#### Σύνταξη `colspan`

```html
<td colspan="2">
```

Η τιμή του είναι ένας θετικός ακέραιος αριθμός.

---

#### Παράδειγμα `colspan`

```html
<table>
    <tr>
        <th>Όνομα</th>
        <th>Ηλικία</th>
        <th>Πόλη</th>
    </tr>

    <tr>
        <td colspan="2">Μαρία - 25</td>
        <td>Αθήνα</td>
    </tr>
</table>
```

---

### Το attribute `rowspan`

Το `rowspan` επιτρέπει σε ένα κελί να εκτείνεται σε περισσότερες από μία γραμμές.

#### Σύνταξη `rowspan`

```html
<td rowspan="2">
```

Η τιμή του είναι ένας θετικός ακέραιος αριθμός.

---

#### Παράδειγμα `rowspan`

```html
<table>
    <tr>
        <th>Όνομα</th>
        <th>Μάθημα</th>
    </tr>

    <tr>
        <td rowspan="2">Μαρία</td>
        <td>Μαθηματικά</td>
    </tr>

    <tr>
        <td>Φυσική</td>
    </tr>
</table>
```

---

## Σημαντικές παρατηρήσεις

- Το `<td>` πρέπει πάντα να βρίσκεται μέσα σε ένα στοιχείο `<tr>`.
- Κάθε γραμμή ενός πίνακα μπορεί να περιέχει ένα ή περισσότερα στοιχεία `<td>`.
- Όλα τα κελιά μιας γραμμής δεν είναι απαραίτητο να έχουν το ίδιο πλάτος.
- Με τα attributes `colspan` και `rowspan` μπορούμε να δημιουργήσουμε πιο σύνθετους πίνακες.

---

## Browser Support

Το tag `<td>` υποστηρίζεται από όλους τους σύγχρονους browsers.

- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari
- Opera
