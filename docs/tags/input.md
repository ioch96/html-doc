# HTML `<input>` Tag

Το tag `<input>` χρησιμοποιείται για τη δημιουργία ενός **πεδίου εισαγωγής δεδομένων (input field)**.

Είναι το σημαντικότερο στοιχείο μιας φόρμας (`<form>`), καθώς επιτρέπει στον χρήστη να εισάγει πληροφορίες, όπως κείμενο, αριθμούς, email, ημερομηνίες, κωδικούς πρόσβασης και πολλά άλλα.

Η συμπεριφορά ενός `<input>` καθορίζεται από το attribute `type`.

Ανάλογα με την τιμή του `type`, το ίδιο στοιχείο μπορεί να εμφανίζεται με διαφορετικό τρόπο και να εξυπηρετεί διαφορετικό σκοπό.

> **Συμβουλή:** Για λόγους προσβασιμότητας, να χρησιμοποιείτε πάντοτε το στοιχείο `<label>` μαζί με τα περισσότερα πεδία `<input>`.

---

## Σύνταξη

```html
<label for="username">
    Όνομα χρήστη:
</label>

<input
    type="text"
    id="username"
    name="username"
>
```

---

## Types

Το `<input>` υποστηρίζει πολλούς διαφορετικούς τύπους.

|Type|Περιγραφή|
|:-:|:-|
|`button`|Απλό κουμπί.|
|`checkbox`|Πλαίσιο επιλογής.|
|`color`|Επιλογή χρώματος.|
|`date`|Επιλογή ημερομηνίας.|
|`datetime-local`|Επιλογή ημερομηνίας και ώρας.|
|`email`|Εισαγωγή email.|
|`file`|Επιλογή αρχείου.|
|`hidden`|Κρυφό πεδίο.|
|`image`|Κουμπί υποβολής με εικόνα.|
|`month`|Επιλογή μήνα.|
|`number`|Εισαγωγή αριθμού.|
|`password`|Εισαγωγή κωδικού πρόσβασης.|
|`radio`|Κουμπί επιλογής.|
|`range`|Ρυθμιστικό (slider).|
|`reset`|Επαναφορά φόρμας.|
|`search`|Πεδίο αναζήτησης.|
|`submit`|Υποβολή φόρμας.|
|`tel`|Εισαγωγή τηλεφώνου.|
|`text`|Απλό πεδίο κειμένου (προεπιλογή).|
|`time`|Επιλογή ώρας.|
|`url`|Εισαγωγή URL.|
|`week`|Επιλογή εβδομάδας.|

> Κάθε τύπος αναλύεται ξεχωριστά στα αντίστοιχα κεφάλαια του βιβλίου.

## Attributes

|Attribute|Περιγραφή|
|:-:|:-|
|`type`|Καθορίζει τον τύπο του πεδίου εισαγωγής.|
|`autocomplete`|Ενεργοποιεί ή απενεργοποιεί την αυτόματη συμπλήρωση.|
|`maxlength`|Ορίζει τον μέγιστο αριθμό χαρακτήρων.|
|`minlength`|Ορίζει τον ελάχιστο αριθμό χαρακτήρων.|
|`pattern`|Ορίζει ένα μοτίβο (Regular Expression) που πρέπει να ακολουθεί η τιμή.|
|`placeholder`|Εμφανίζει προσωρινό κείμενο μέσα στο πεδίο.|
|`required`|Καθιστά το πεδίο υποχρεωτικό.|

---

## Το attribute `type`

Καθορίζει τον τύπο του πεδίου εισαγωγής.

```html
<input type="text">
```

```html
<input type="email">
```

```html
<input type="password">
```

---

## Το attribute `autocomplete`

Επιτρέπει στον browser να προτείνει προηγούμενες τιμές.

```html
<input
    type="email"
    autocomplete="email"
>
```

ή

```html
<input
    type="text"
    autocomplete="off"
>
```

---

## Το attribute `maxlength`

Ορίζει τον μέγιστο αριθμό χαρακτήρων.

```html
<input
    type="text"
    maxlength="20"
>
```

Ο χρήστης δεν μπορεί να πληκτρολογήσει περισσότερους από 20 χαρακτήρες.

---

## Το attribute `minlength`

Ορίζει τον ελάχιστο αριθμό χαρακτήρων.

```html
<input
    type="password"
    minlength="8"
>
```

Η φόρμα δεν μπορεί να υποβληθεί αν ο χρήστης εισαγάγει λιγότερους από 8 χαρακτήρες.

---

## Το attribute `pattern`

Καθορίζει ένα μοτίβο που πρέπει να ακολουθεί η τιμή.

```html
<input
    type="text"
    pattern="[A-Za-z]{3,10}"
>
```

Στο παραπάνω παράδειγμα επιτρέπονται μόνο λατινικά γράμματα, από 3 έως 10 χαρακτήρες.

---

## Το attribute `placeholder`

Εμφανίζει ένα προσωρινό κείμενο μέσα στο πεδίο.

```html
<input
    type="email"
    placeholder="example@email.com"
>
```

Το κείμενο εξαφανίζεται μόλις ο χρήστης αρχίσει να πληκτρολογεί.

---

## Το attribute `required`

Καθιστά το πεδίο υποχρεωτικό.

```html
<input
    type="text"
    required
>
```

Η φόρμα δεν μπορεί να υποβληθεί αν το πεδίο παραμείνει κενό.

---

## Σημαντικές παρατηρήσεις

- Το `<input>` είναι το βασικότερο στοιχείο για την εισαγωγή δεδομένων σε μια φόρμα.
- Η λειτουργία του εξαρτάται από το attribute `type`.
- Το attribute `name` είναι απαραίτητο όταν θέλουμε να αποσταλεί η τιμή του πεδίου κατά την υποβολή της φόρμας.
- Το `id` χρησιμοποιείται για τη σύνδεση του `<input>` με το αντίστοιχο `<label>`.
- Τα attributes `required`, `maxlength`, `minlength` και `pattern` βοηθούν στην επικύρωση των δεδομένων πριν από την υποβολή της φόρμας.

> **Βέλτιστη πρακτική:** Να χρησιμοποιείτε πάντοτε ένα `<label>` για κάθε στοιχείο `<input>` και να ορίζετε τα attributes `id`, `name` και `type`, ώστε η φόρμα να είναι προσβάσιμη, σωστά δομημένη και εύκολη στη χρήση.
