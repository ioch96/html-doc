# HTML `<img>` Tag

Το tag `<img>` χρησιμοποιείται για την **ενσωμάτωση μιας εικόνας (image)** σε μια ιστοσελίδα HTML.

Οι εικόνες **δεν αποθηκεύονται μέσα στο HTML έγγραφο**. Αντίθετα, το `<img>` δημιουργεί μια θέση (placeholder) και **συνδέεται** με το αρχείο της εικόνας μέσω του attribute `src`.

Το στοιχείο `<img>` είναι ένα **κενό στοιχείο (empty element)**, επομένως **δεν διαθέτει tag κλεισίματος**.

## Υποχρεωτικά Attributes

Το `<img>` διαθέτει δύο υποχρεωτικά attributes:

- `src` — Καθορίζει τη διαδρομή (path) ή τη διεύθυνση (URL) της εικόνας.
- `alt` — Παρέχει εναλλακτικό κείμενο που εμφανίζεται όταν η εικόνα δεν μπορεί να φορτωθεί και χρησιμοποιείται επίσης από τις τεχνολογίες υποβοήθησης (screen readers).

> **Σημείωση:** Συνιστάται να ορίζετε πάντα τα attributes `width` και `height`. Έτσι ο browser γνωρίζει εκ των προτέρων τις διαστάσεις της εικόνας και αποφεύγονται μετακινήσεις του περιεχομένου κατά τη φόρτωση της σελίδας (layout shift).
>
> **Συμβουλή:** Αν θέλετε η εικόνα να λειτουργεί ως σύνδεσμος, τοποθετήστε το `<img>` μέσα σε ένα στοιχείο `<a>`.

## Σύνταξη

```html
<img src="image.jpg" alt="Περιγραφή εικόνας" width="300" height="200">
```

## Attributes

|Attribute|Τιμή|Περιγραφή|
|:---------:|:----:|-----------|
|`src`|URL|Καθορίζει τη διαδρομή ή τη διεύθυνση της εικόνας.|
|`alt`|text|Παρέχει εναλλακτικό κείμενο για την εικόνα.|
|`width`|pixels|Specifies the width of an image|
|`height`|pixels|Specifies the height of an image|

## Παράδειγμα

```html
<img src="mountains.jpg" alt="Βουνά κατά το ηλιοβασίλεμα" width="500" height="333">
```

## Το attribute `src`

Το `src` καθορίζει από πού θα φορτωθεί η εικόνα.

Μπορεί να είναι:

### Σχετική διαδρομή (Relative Path)

```html
<img src="images/logo.png" alt="Logo">
```

### Απόλυτη διεύθυνση (Absolute URL)

```html
<img src="https://example.com/images/logo.png" alt="Logo">
```

## Το attribute `alt`

Το `alt` περιέχει μια περιγραφή της εικόνας.

Χρησιμοποιείται όταν:

- Η εικόνα δεν μπορεί να φορτωθεί.
- Ο χρήστης χρησιμοποιεί screen reader.
- Οι μηχανές αναζήτησης αναλύουν το περιεχόμενο της σελίδας.

Παράδειγμα:

```html
<img src="cat.jpg" alt="Μια γάτα που κοιμάται πάνω σε έναν καναπέ">
```

Αν η εικόνα δεν εμφανιστεί, ο browser θα εμφανίσει το κείμενο του `alt`.

## Διαστάσεις εικόνας

Συνιστάται να ορίζετε πάντα τις διαστάσεις.

```html
<img src="photo.jpg" alt="Παράδειγμα" width="400" height="300">
```

Έτσι ο browser δεσμεύει τον απαραίτητο χώρο πριν φορτωθεί η εικόνα.

## Εικόνα ως σύνδεσμος

Μπορείτε να τοποθετήσετε ένα `<img>` μέσα σε ένα `<a>`.

```html
<a href="https://example.com"><img src="logo.png" alt="Logo" width="120" height="120"></a>
```

Όταν ο χρήστης κάνει κλικ στην εικόνα, ανοίγει ο σύνδεσμος.

## Βέλτιστες Πρακτικές (Best Practices)

- Χρησιμοποιείτε πάντα το attribute `alt`.
- Ορίζετε πάντα `width` και `height`, όταν γνωρίζετε τις πραγματικές διαστάσεις της εικόνας.
- Χρησιμοποιείτε εικόνες κατάλληλου μεγέθους για καλύτερη απόδοση της ιστοσελίδας.
- Αποφεύγετε να χρησιμοποιείτε εικόνες αντί για κείμενο, εκτός αν είναι απαραίτητο.

> **Σημείωση:** Το `<img>` είναι ένα από τα σημαντικότερα στοιχεία της HTML και χρησιμοποιείται σχεδόν σε κάθε ιστοσελίδα. Τα δύο σημαντικότερα attributes που πρέπει να θυμάστε είναι το **`src`**, που καθορίζει τη θέση της εικόνας, και το **`alt`**, που βελτιώνει την προσβασιμότητα και το SEO.
