# HTML `<audio>` Tag

Το tag `<audio>` χρησιμοποιείται για την **ενσωμάτωση αρχείων ήχου** σε μια ιστοσελίδα.

Το `<audio>` είναι ένα **σημασιολογικό στοιχείο (semantic element)** και μπορεί να περιέχει ένα ή περισσότερα στοιχεία `<source>`. Ο browser επιλέγει αυτόματα την **πρώτη μορφή ήχου που υποστηρίζει**.

Το κείμενο που βρίσκεται ανάμεσα στα tags `<audio>` και `</audio>` εμφανίζεται μόνο σε browsers που **δεν υποστηρίζουν** το στοιχείο `<audio>`.

---

## Υποστηριζόμενες μορφές ήχου

Οι πιο συνηθισμένες μορφές ήχου στην HTML είναι:

- **MP3** (`.mp3`)
- **WAV** (`.wav`)
- **OGG** (`.ogg`)

|Browser|MP3|WAV|OGG|
|:-------:|:---:|:---:|:---:|
|Edge|🟢|🟢|🟢|
|Chrome|🟢|🟢|🟢|
|Firefox|🟢|🟢|🟢|
|Safari|🟢|🟢|🔴|
|Opera|🟢|🟢|🟢|

> **Σημείωση:** Η μορφή **MP3** προσφέρει τη μεγαλύτερη συμβατότητα και είναι η προτεινόμενη επιλογή.

---

## Σύνταξη

```html
<audio controls>
  <source src="music.mp3" type="audio/mpeg">
  Ο browser σας δεν υποστηρίζει το στοιχείο audio.
</audio>
```

---

## Χρήση πολλαπλών μορφών

Μπορείτε να προσθέσετε περισσότερα από ένα στοιχεία `<source>`.

```html
<audio controls>
  <source src="music.mp3" type="audio/mpeg">
  <source src="music.ogg" type="audio/ogg">
  <source src="music.wav" type="audio/wav">
  Ο browser σας δεν υποστηρίζει το στοιχείο audio.
</audio>
```

Ο browser θα χρησιμοποιήσει την **πρώτη μορφή** που μπορεί να αναπαράγει.

## Attributes

|Attribute|Τιμή|Περιγραφή|
|:---------:|:----:|-----------|
|`autoplay`|`autoplay`|Ξεκινά αυτόματα την αναπαραγωγή όταν φορτώσει το αρχείο.|
|`controls`|`controls`|Εμφανίζει τα χειριστήρια αναπαραγωγής (Play, Pause, Volume).|
|`loop`|`loop`|Επαναλαμβάνει συνεχώς τον ήχο.|
|`muted`|`muted`|Ξεκινά τον ήχο χωρίς ένταση (σιωπηλά).|
|`preload`|`auto`, `metadata`, `none`|Καθορίζει τον τρόπο φόρτωσης του αρχείου ήχου.|
|`src`|URL|Καθορίζει τη διαδρομή (URL) του αρχείου ήχου.|

> **Σημείωση:** Οι περισσότεροι σύγχρονοι browsers επιτρέπουν το `autoplay` μόνο όταν ο ήχος είναι **σιωπηλός (`muted`)** ή μετά από αλληλεπίδραση του χρήστη με τη σελίδα.

### Τιμές του attribute `preload`

|Τιμή|Περιγραφή|
|------|-----------|
|`none`|Δεν φορτώνει τίποτα μέχρι να πατήσει ο χρήστης Play.|
|`metadata`|Φορτώνει μόνο βασικές πληροφορίες (διάρκεια, τίτλος κ.λπ.).|
|`auto`|Ο browser αποφασίζει πόσο θα φορτώσει.|

---

## Best Practices

- Προτιμήστε τη μορφή **MP3** για μέγιστη συμβατότητα.
- Χρησιμοποιείτε πάντα το attribute `controls`, ώστε ο χρήστης να μπορεί να ελέγχει την αναπαραγωγή.
- Αν θέλετε να υποστηρίζονται πολλοί browsers, προσθέστε περισσότερα από ένα στοιχεία `<source>`.

> **Σημείωση:** Το `<audio>` είναι το σύγχρονο HTML στοιχείο για την αναπαραγωγή ήχου. Χρησιμοποιείται ευρέως σε σύγχρονες ιστοσελίδες και εφαρμογές για την ενσωμάτωση μουσικής, podcast και άλλου ηχητικού περιεχομένου.
