# HTML `<embed>` Tag

Το tag `<embed>` χρησιμοποιείται για την **ενσωμάτωση (embedding)** εξωτερικού περιεχομένου μέσα σε μια ιστοσελίδα.

Μπορεί να χρησιμοποιηθεί για την εμφάνιση:

- Αρχείων PDF
- Εικόνων
- Βίντεο
- Ήχου
- GIF
- Άλλων αρχείων πολυμέσων
- Περιεχομένου από εξωτερικές εφαρμογές

Το `<embed>` είναι **self-closing tag**, δηλαδή **δεν έχει tag κλεισίματος**.

```html
<embed src="example.pdf">
```

---

## Προειδοποίηση

Στο παρελθόν το `<embed>` χρησιμοποιούνταν για την ενσωμάτωση:

- Java Applets
- Flash (Shockwave Flash)
- ActiveX
- Browser Plug-ins

Σήμερα όμως:

- Οι **Java Applets** δεν υποστηρίζονται πλέον.
- Το **Flash** έχει καταργηθεί.
- Το **ActiveX** δεν υποστηρίζεται από κανέναν σύγχρονο browser.

Για αυτόν τον λόγο, η χρήση του `<embed>` είναι σήμερα αρκετά περιορισμένη.

---

## Προτεινόμενες εναλλακτικές

Ανάλογα με το περιεχόμενο, προτιμήστε τα κατάλληλα HTML στοιχεία:

|Περιεχόμενο|Προτεινόμενο στοιχείο|
|-------------|-----------------------|
|Εικόνα|`<img>`|
|Βίντεο|`<video>`|
|Ήχος|`<audio>`|
|Ιστοσελίδα ή HTML έγγραφο|`<iframe>`|

Τα παραπάνω στοιχεία είναι πιο σύγχρονα, πιο σημασιολογικά και υποστηρίζονται καλύτερα.

---

## Σύνταξη

```html
<embed
  src="example.pdf"
  type="application/pdf"
  width="600"
  height="500"
>
```

---

## Παράδειγμα

```html
<!DOCTYPE html>
<html lang="el">

<head>
  <title>Το στοιχείο embed</title>
</head>

<body>

  <h1>Ενσωμάτωση PDF</h1>

  <embed
    src="document.pdf"
    type="application/pdf"
    width="600"
    height="500"
  >

</body>

</html>
```

---

## Παράδειγμα με GIF

```html
<embed
  src="animation.gif"
  width="300"
  height="200"
>
```

---

## Attributes

|Attribute|Τιμή|Περιγραφή|
|:---------:|:----:|-----------|
|`src`|URL|Καθορίζει τη διαδρομή (URL) του αρχείου που θα ενσωματωθεί.|
|`type`|MIME Type|Καθορίζει τον τύπο του αρχείου (π.χ. `application/pdf`).|
|`width`|pixels|Καθορίζει το πλάτος του περιεχομένου.|
|`height`|pixels|Καθορίζει το ύψος του περιεχομένου.|

---

## Το attribute `type`

Το `type` βοηθά τον browser να καταλάβει τι είδους αρχείο πρόκειται να εμφανίσει.

Παράδειγμα:

```html
<embed
  src="document.pdf"
  type="application/pdf"
>
```

Άλλα παραδείγματα:

```html
type="image/svg+xml"
```

```html
type="application/pdf"
```

---

## Βέλτιστες Πρακτικές (Best Practices)

- Χρησιμοποιείτε το `<embed>` μόνο όταν χρειάζεται να ενσωματώσετε περιεχόμενο που δεν καλύπτεται από άλλα πιο εξειδικευμένα στοιχεία.
- Για εικόνες χρησιμοποιείτε το `<img>`.
- Για βίντεο χρησιμοποιείτε το `<video>`.
- Για ήχο χρησιμοποιείτε το `<audio>`.
- Για ιστοσελίδες ή HTML έγγραφα χρησιμοποιείτε το `<iframe>`.
- Όταν χρησιμοποιείτε το `<embed>`, ορίζετε πάντα τα `width` και `height` για καλύτερη εμφάνιση.

> **Σημείωση:** Παρότι το `<embed>` εξακολουθεί να υποστηρίζεται από τους περισσότερους σύγχρονους browsers, χρησιμοποιείται πολύ λιγότερο από παλαιότερα. Στις περισσότερες περιπτώσεις προτιμώνται τα εξειδικευμένα στοιχεία της HTML, όπως `<img>`, `<audio>`, `<video>` και `<iframe>`, τα οποία προσφέρουν καλύτερη συμβατότητα, προσβασιμότητα και σημασιολογία.
