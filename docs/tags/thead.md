# HTML `<thead>` Tag

Το tag `<thead>` χρησιμοποιείται για να **ομαδοποιεί την κεφαλίδα (header)** ενός πίνακα HTML.

Η κεφαλίδα ενός πίνακα περιέχει συνήθως τις επικεφαλίδες των στηλών, δηλαδή τα στοιχεία `<th>` που περιγράφουν τα δεδομένα του πίνακα.

Το `<thead>` δεν δημιουργεί από μόνο του κάποια νέα γραμμή ή κελί. Λειτουργεί ως **container** που οργανώνει το τμήμα της κεφαλίδας του πίνακα.

---

## Σύνταξη

```html
<thead>
    ...
</thead>
```

---

## Παράδειγμα

```html
<table>
    <thead>
        <tr>
            <th>Όνομα</th>
            <th>Ηλικία</th>
            <th>Πόλη</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Μαρία</td>
            <td>25</td>
            <td>Αθήνα</td>
        </tr>
        <tr>
            <td>Γιώργος</td>
            <td>31</td>
            <td>Θεσσαλονίκη</td>
        </tr>
    </tbody>
</table>
```

---

## Τι κάνει το `<thead>`

Το `<thead>` ομαδοποιεί όλες τις γραμμές που αποτελούν την **κεφαλίδα του πίνακα**.

Συνήθως περιέχει μία γραμμή (`<tr>`), η οποία περιέχει τα κελιά επικεφαλίδας (`<th>`).

Παράδειγμα:

```html
<thead>
    <tr>
        <th>Όνομα</th>
        <th>Ηλικία</th>
    </tr>
</thead>
```

Όλα τα στοιχεία μέσα στο `<thead>` θεωρούνται μέρος της κεφαλίδας του πίνακα.

---

## Το `<thead>` πρέπει να περιέχει `<tr>`

Το `<thead>` δεν μπορεί να περιέχει απευθείας στοιχεία `<th>` ή `<td>`.

Πρέπει να περιέχει τουλάχιστον ένα στοιχείο `<tr>`.

Σωστό:

```html
<thead>
    <tr>
        <th>Όνομα</th>
        <th>Ηλικία</th>
    </tr>
</thead>
```

Λάθος:

```html
<thead>
    <th>Όνομα</th>
    <th>Ηλικία</th>
</thead>
```

Τα `<th>` πρέπει πάντα να βρίσκονται μέσα σε ένα `<tr>`.

---

## Το `<thead>` χρησιμοποιείται μαζί με τα `<tbody>` και `<tfoot>`

Ένας πίνακας μπορεί να χωριστεί σε τρία βασικά τμήματα:

- `<thead>` &rarr; κεφαλίδα του πίνακα.
- `<tbody>` &rarr; κύριο σώμα του πίνακα.
- `<tfoot>` &rarr; υποσέλιδο του πίνακα.

Παράδειγμα:

```html
<table>
    <thead>
        ...
    </thead>

    <tbody>
        ...
    </tbody>

    <tfoot>
        ...
    </tfoot>
</table>
```

Αυτή η οργάνωση κάνει τον πίνακα πιο ευανάγνωστο και πιο εύκολο στη συντήρηση.

---

## Επηρεάζει την εμφάνιση του πίνακα

Όχι.

Το `<thead>` **δεν αλλάζει από μόνο του την εμφάνιση** του πίνακα.

Δημιουργεί μόνο μια λογική ομαδοποίηση της κεφαλίδας.

Η μορφοποίηση της κεφαλίδας γίνεται συνήθως με CSS.

---

## Γιατί χρησιμοποιούμε το `<thead>`

Παρότι ένας πίνακας μπορεί να λειτουργήσει και χωρίς `<thead>`, η χρήση του προσφέρει αρκετά πλεονεκτήματα.

### Καλύτερη οργάνωση

Διαχωρίζει καθαρά:

- την κεφαλίδα,
- τα δεδομένα,
- το υποσέλιδο.

Έτσι ο κώδικας γίνεται πιο ευανάγνωστος.

---

### Καλύτερη σημασιολογία

Το `<thead>` δηλώνει ξεκάθαρα ότι το συγκεκριμένο τμήμα αποτελεί την κεφαλίδα του πίνακα.

Αυτό βοηθά:

- τους browsers,
- τις μηχανές αναζήτησης,
- τα βοηθητικά προγράμματα προσβασιμότητας (screen readers).

---

### Μεγάλοι πίνακες

Σε μεγάλους πίνακες, οι browsers ή άλλα προγράμματα μπορούν να αξιοποιήσουν το `<thead>` ώστε:

- η κεφαλίδα να παραμένει ορατή κατά την κύλιση του πίνακα,
- να επαναλαμβάνεται στην αρχή κάθε σελίδας κατά την εκτύπωση ενός μεγάλου πίνακα.

Η συμπεριφορά αυτή εξαρτάται από τον browser ή από το CSS που χρησιμοποιείται.

---

## Υποχρεωτικό tag κλεισίματος

Το `<thead>` πρέπει να κλείνει με το αντίστοιχο tag.

Σωστό:

```html
<thead>

</thead>
```

Λάθος:

```html
<thead>
```

---

## Browser Support

Το tag `<thead>` υποστηρίζεται από όλους τους σύγχρονους browsers.

- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari
- Opera
