# HTML `<body>` Tag

Το tag `<body>` ορίζει το **σώμα (body)** ενός εγγράφου HTML.

Το στοιχείο `<body>` περιέχει **όλο το ορατό περιεχόμενο** μιας ιστοσελίδας, όπως:

- Επικεφαλίδες
- Παραγράφους
- Εικόνες
- Υπερσυνδέσμους
- Πίνακες
- Λίστες
- Φόρμες
- Κουμπιά
- Και γενικά όλα τα στοιχεία που εμφανίζονται στον χρήστη.

> **Σημείωση:** Κάθε έγγραφο HTML μπορεί να περιέχει **μόνο ένα** στοιχείο `<body>`.

## Παράδειγμα

```html
<!DOCTYPE html>
<html lang="el">
  <head>
    <title>Τίτλος του Εγγράφου</title>
  </head>
  <body>
    <h1>Αυτή είναι μια επικεφαλίδα</h1>
    <p>Αυτή είναι μια παράγραφος.</p>
  </body>
</html>
```

> **Σημείωση:** Το `<body>` βρίσκεται πάντα μετά το στοιχείο `<head>`. Ό,τι τοποθετείται μέσα στο `<head>` (όπως ο τίτλος της σελίδας, τα CSS και τα metadata) **δεν εμφανίζεται** στο περιεχόμενο της ιστοσελίδας, ενώ ό,τι βρίσκεται μέσα στο `<body>` είναι αυτό που βλέπει ο χρήστης στον browser.
