# HTML `<sub>` Tag

Το tag `<sub>` χρησιμοποιείται για την εμφάνιση **δείκτη (subscript)**. Το περιεχόμενό του εμφανίζεται ελαφρώς **κάτω από τη βασική γραμμή του κειμένου** και συνήθως με μικρότερο μέγεθος γραμματοσειράς.

Το στοιχείο `<sub>` χρησιμοποιείται κυρίως για:

- χημικούς τύπους,
- μαθηματικές παραστάσεις,
- δείκτες μεταβλητών,
- επιστημονικούς συμβολισμούς.

## Παράδειγμα 1

Χρήση σε χημικό τύπο.

```html
<p>H<sub>2</sub>O</p>
```

---

## Παράδειγμα 2

Χρήση σε χημικό τύπο.

```html
<p>CO<sub>2</sub></p>
```

---

## Παράδειγμα 3

Χρήση σε μαθηματική παράσταση.

```html
<p>x<sub>1</sub> + x<sub>2</sub> = x<sub>3</sub></p>
```

---

## Παράδειγμα 4

Χρήση σε επιστημονικό συμβολισμό.

```html
<p>Η μεταβλητή V<sub>max</sub> αντιπροσωπεύει τη μέγιστη ταχύτητα.</p>
```
