# HTML `<dl>` Tag

Το tag `<dl>` χρησιμοποιείται για τη δημιουργία μιας **λίστας περιγραφών (description list)**.

Σε μια λίστα περιγραφών, κάθε στοιχείο αποτελείται από:

- ένα ή περισσότερα στοιχεία `<dt>` (**Description Term**) που ορίζουν τον όρο ή το όνομα,
- ένα ή περισσότερα στοιχεία `<dd>` (**Description Details**) που περιέχουν την περιγραφή του όρου.

Το `<dl>` χρησιμοποιείται όταν θέλουμε να παρουσιάσουμε **όρους και τις αντίστοιχες περιγραφές τους**, αντί για μια απλή αριθμημένη ή μη αριθμημένη λίστα.

## Σύνταξη

```html
<dl>
  <dt>Όρος</dt>
  <dd>Περιγραφή του όρου.</dd>
</dl>
```

## Παράδειγμα

```html
<dl>
  <dt>HTML</dt>
  <dd>Γλώσσα σήμανσης για τη δημιουργία ιστοσελίδων.</dd>

  <dt>CSS</dt>
  <dd>Γλώσσα μορφοποίησης ιστοσελίδων.</dd>

  <dt>JavaScript</dt>
  <dd>Γλώσσα προγραμματισμού για διαδραστικές ιστοσελίδες.</dd>
</dl>
```

## Πολλαπλές περιγραφές για τον ίδιο όρο

Ένας όρος μπορεί να έχει περισσότερες από μία περιγραφές.

```html
<dl>
  <dt>HTML</dt>
  <dd>Γλώσσα σήμανσης.</dd>
  <dd>Χρησιμοποιείται για τη δημιουργία ιστοσελίδων.</dd>
</dl>
```

## Πολλοί όροι με την ίδια περιγραφή

Είναι επίσης δυνατό πολλοί όροι να μοιράζονται την ίδια περιγραφή.

```html
<dl>
  <dt>TV</dt>
  <dt>Τηλεόραση</dt>
  <dd>Ηλεκτρονική συσκευή προβολής εικόνας και ήχου.</dd>
</dl>
```

## Βέλτιστες Πρακτικές (Best Practices)

- Χρησιμοποιείτε το `<dl>` όταν παρουσιάζετε **όρους και τις επεξηγήσεις τους**.
- Χρησιμοποιείτε τα στοιχεία `<dt>` μόνο για τον όρο ή το όνομα.
- Χρησιμοποιείτε τα στοιχεία `<dd>` μόνο για την περιγραφή του όρου.
- Μην χρησιμοποιείτε το `<dl>` για τη δημιουργία απλών λιστών. Σε αυτήν την περίπτωση προτιμήστε τα στοιχεία `<ul>` ή `<ol>`.

> **Σημείωση:** Το `<dl>` είναι ένα **σημασιολογικό στοιχείο (semantic element)**. Δηλώνει ότι το περιεχόμενο αποτελείται από όρους και τις αντίστοιχες περιγραφές τους, γεγονός που βοηθά τους browsers, τις μηχανές αναζήτησης και τις τεχνολογίες υποβοήθησης να κατανοήσουν καλύτερα τη δομή του περιεχομένου.
