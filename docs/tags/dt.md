# HTML `<dt>` Tag

Το tag `<dt>` (**Description Term**) χρησιμοποιείται για να ορίσει έναν **όρο ή όνομα** μέσα σε μια **λίστα περιγραφών (description list)**.

Το `<dt>` χρησιμοποιείται πάντα μαζί με:

- το `<dl>`, το οποίο ορίζει τη λίστα περιγραφών,
- το `<dd>`, το οποίο περιέχει την περιγραφή του όρου.

## Σύνταξη

```html
<dl>
  <dt>Όρος</dt>
  <dd>Περιγραφή του όρου.</dd>
</dl>
```

## Παράδειγμα

```html
<dl>
  <dt>HTML</dt>
  <dd>Γλώσσα σήμανσης για τη δημιουργία ιστοσελίδων.</dd>

  <dt>CSS</dt>
  <dd>Γλώσσα μορφοποίησης ιστοσελίδων.</dd>

  <dt>JavaScript</dt>
  <dd>Γλώσσα προγραμματισμού για διαδραστικές ιστοσελίδες.</dd>
</dl>
```

## Πολλοί όροι με την ίδια περιγραφή

Μπορείτε να χρησιμοποιήσετε περισσότερα από ένα στοιχεία `<dt>` πριν από ένα `<dd>`.

```html
<dl>
  <dt>TV</dt>
  <dt>Τηλεόραση</dt>
  <dd>Ηλεκτρονική συσκευή προβολής εικόνας και ήχου.</dd>
</dl>
```

## Βέλτιστες Πρακτικές (Best Practices)

- Χρησιμοποιείτε το `<dt>` μόνο μέσα σε ένα στοιχείο `<dl>`.
- Το `<dt>` πρέπει να περιέχει μόνο τον όρο ή το όνομα και όχι την περιγραφή του.
- Χρησιμοποιείτε το `<dd>` για την περιγραφή που αντιστοιχεί στον όρο.
- Ένα στοιχείο `<dt>` μπορεί να αντιστοιχεί σε ένα ή περισσότερα στοιχεία `<dd>`.

> **Σημείωση:** Το `<dt>` είναι ένα **σημασιολογικό στοιχείο (semantic element)**. Δηλώνει ότι το περιεχόμενό του αποτελεί έναν όρο ή όνομα που θα περιγραφεί από το αντίστοιχο στοιχείο `<dd>`.
