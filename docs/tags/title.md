# HTML `<title>` Tag

Το tag `<title>` ορίζει τον **τίτλο ενός εγγράφου HTML**. Ο τίτλος μπορεί να περιέχει **μόνο κείμενο** και εμφανίζεται:

- Στην **καρτέλα (tab)** του browser.
- Στη **γραμμή τίτλου (title bar)** του browser (όπου υποστηρίζεται).
- Στα **αποτελέσματα των μηχανών αναζήτησης** (όπως η Google).

Το στοιχείο `<title>` είναι **υποχρεωτικό** σε κάθε έγγραφο HTML και πρέπει να βρίσκεται μέσα στο στοιχείο `<head>`.

## Σύνταξη

```html
<head>
  <title>Τίτλος Σελίδας</title>
</head>
```

## Παράδειγμα

```html
<!DOCTYPE html>
<html lang="el">
<head>
  <title>Αναφορά Στοιχείων HTML</title>
</head>
<body>
  <h1>Αυτή είναι μια επικεφαλίδα</h1>
  <p>Αυτή είναι μια παράγραφος.</p>
</body>
</html>
```

## Γιατί είναι σημαντικό το tag `<title>`

Το στοιχείο `<title>` εξυπηρετεί πολλούς σημαντικούς σκοπούς:

- Εμφανίζει τον τίτλο της σελίδας στην καρτέλα (tab) ή στη γραμμή τίτλου του browser.
- Χρησιμοποιείται ως τίτλος όταν η σελίδα αποθηκεύεται στους **σελιδοδείκτες (bookmarks)** ή στα **Αγαπημένα (Favorites)**.
- Εμφανίζεται ως ο **τίτλος (clickable headline)** στα αποτελέσματα των μηχανών αναζήτησης.
- Βοηθά τις μηχανές αναζήτησης να κατανοήσουν το περιεχόμενο της σελίδας, γεγονός που το καθιστά σημαντικό για τη **Βελτιστοποίηση για Μηχανές Αναζήτησης (Search Engine Optimization - SEO)**.
- Βελτιώνει την εμπειρία του χρήστη, καθώς διευκολύνει την αναγνώριση της σελίδας όταν υπάρχουν πολλές ανοιχτές καρτέλες.

## Best Practices

- Χρησιμοποιείτε **μοναδικό τίτλο** για κάθε σελίδα.
- Ο τίτλος πρέπει να είναι **σαφής**, **περιγραφικός** και **σχετικός** με το περιεχόμενο της σελίδας.
- Διατηρήστε τον τίτλο σχετικά σύντομο, ώστε να μην περικόπτεται στα αποτελέσματα των μηχανών αναζήτησης.
