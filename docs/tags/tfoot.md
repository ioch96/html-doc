# HTML `<tfoot>` Tag

Το tag `<tfoot>` χρησιμοποιείται για να **ομαδοποιεί το υποσέλιδο (footer)** ενός πίνακα HTML.

Το υποσέλιδο ενός πίνακα περιέχει συνήθως πληροφορίες που σχετίζονται με ολόκληρο τον πίνακα, όπως:

- σύνολα,
- μέσους όρους,
- τελικά αποτελέσματα,
- παρατηρήσεις,
- συμπεράσματα.

Το `<tfoot>` λειτουργεί ως **container** για τις γραμμές που αποτελούν το υποσέλιδο του πίνακα.

---

## Σύνταξη

```html
<tfoot>
    ...
</tfoot>
```

---

## Παράδειγμα

```html
<table>
    <thead>
        <tr>
            <th>Προϊόν</th>
            <th>Ποσότητα</th>
            <th>Τιμή</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Laptop</td>
            <td>2</td>
            <td>900€</td>
        </tr>
        <tr>
            <td>Οθόνη</td>
            <td>3</td>
            <td>250€</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <th colspan="2">Συνολικό Κόστος</th>
            <td>2550€</td>
        </tr>
    </tfoot>
</table>
```

---

## Τι κάνει το `<tfoot>`

Το `<tfoot>` ομαδοποιεί όλες τις γραμμές που αποτελούν το υποσέλιδο του πίνακα.

Συνήθως περιέχει μία ή περισσότερες γραμμές (`<tr>`), μέσα στις οποίες υπάρχουν στοιχεία `<th>` ή `<td>`.

Παράδειγμα:

```html
<tfoot>
    <tr>
        <th colspan="2">Σύνολο</th>
        <td>1500€</td>
    </tr>
</tfoot>
```

Όλα τα στοιχεία μέσα στο `<tfoot>` θεωρούνται μέρος του υποσέλιδου του πίνακα.

---

## Το `<tfoot>` πρέπει να περιέχει `<tr>`

Το `<tfoot>` δεν μπορεί να περιέχει απευθείας στοιχεία `<th>` ή `<td>`.

Πρέπει να περιέχει τουλάχιστον ένα στοιχείο `<tr>`.

Σωστό:

```html
<tfoot>
    <tr>
        <td>Σύνολο</td>
    </tr>
</tfoot>
```

Λάθος:

```html
<tfoot>
    <td>Σύνολο</td>
</tfoot>
```

Τα στοιχεία `<th>` και `<td>` πρέπει πάντα να βρίσκονται μέσα σε ένα `<tr>`.

---

## Το `<tfoot>` χρησιμοποιείται μαζί με τα `<thead>` και `<tbody>`

Ένας πίνακας μπορεί να χωριστεί σε τρία βασικά τμήματα:

- `<thead>` → κεφαλίδα του πίνακα.
- `<tbody>` → κύριο σώμα του πίνακα.
- `<tfoot>` → υποσέλιδο του πίνακα.

Παράδειγμα:

```html
<table>
    <thead>
        ...
    </thead>
    <tbody>
        ...
    </tbody>
    <tfoot>
        ...
    </tfoot>
</table>
```

Αυτός ο διαχωρισμός κάνει τον πίνακα πιο οργανωμένο και πιο εύκολο στη συντήρηση.

---

## Είναι υποχρεωτικό το `<tfoot>`

Όχι.

Το `<tfoot>` χρησιμοποιείται μόνο όταν ο πίνακας διαθέτει πληροφορίες που ανήκουν στο υποσέλιδο.

Αν ο πίνακας περιέχει μόνο επικεφαλίδες και δεδομένα, δεν υπάρχει λόγος να χρησιμοποιηθεί.

Παράδειγμα πίνακα χωρίς `<tfoot>`:

```html
<table>
    <thead>
        ...
    </thead>
    <tbody>
        ...
    </tbody>
</table>
```

---

## Επηρεάζει την εμφάνιση του πίνακα

Όχι.

Το `<tfoot>` **δεν αλλάζει από μόνο του την εμφάνιση** του πίνακα.

Η μορφοποίηση του υποσέλιδου γίνεται συνήθως με CSS.

---

## Γιατί χρησιμοποιούμε το `<tfoot>`;

Η χρήση του `<tfoot>` προσφέρει αρκετά πλεονεκτήματα.

### Καλύτερη οργάνωση

Διαχωρίζει καθαρά:

- την κεφαλίδα,
- τα δεδομένα,
- το υποσέλιδο.

---

### Καλύτερη σημασιολογία

Δηλώνει ότι το συγκεκριμένο τμήμα περιέχει πληροφορίες που αφορούν το σύνολο του πίνακα.

Αυτό βοηθά:

- τους browsers,
- τις μηχανές αναζήτησης,
- τα βοηθητικά προγράμματα προσβασιμότητας (screen readers).

---

### Μεγάλοι πίνακες

Σε μεγάλους πίνακες, οι browsers ή άλλα προγράμματα μπορούν να αξιοποιήσουν το `<tfoot>` ώστε:

- να εμφανίζουν ή να διαχειρίζονται ξεχωριστά το υποσέλιδο,
- να επαναλαμβάνουν το υποσέλιδο κατά την εκτύπωση πινάκων που εκτείνονται σε πολλές σελίδες.

Η συμπεριφορά αυτή εξαρτάται από τον browser ή από το CSS που χρησιμοποιείται.

---

## Υποχρεωτικό tag κλεισίματος

Το `<tfoot>` πρέπει να κλείνει με το αντίστοιχο tag.

Σωστό:

```html
<tfoot>

</tfoot>
```

Λάθος:

```html
<tfoot>
```

---

## Browser Support

Το tag `<tfoot>` υποστηρίζεται από όλους τους σύγχρονους browsers.

- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari
- Opera
