# HTML `<button>` Tag

Το tag `<button>` χρησιμοποιείται για τη δημιουργία ενός **κουμπιού (button)** σε μια ιστοσελίδα.

Ένα κουμπί μπορεί να χρησιμοποιηθεί για:

- αποστολή μιας φόρμας,
- επαναφορά των πεδίων μιας φόρμας,
- εκτέλεση ενεργειών με JavaScript,
- πλοήγηση ή άλλες λειτουργίες μιας εφαρμογής.

Σε αντίθεση με το `<input type="button">`, το `<button>` μπορεί να περιέχει όχι μόνο κείμενο αλλά και άλλα HTML στοιχεία, όπως εικόνες, εικονίδια και στοιχεία μορφοποίησης.

---

## Σύνταξη

```html
<button type="button">Κείμενο κουμπιού</button>
```

---

## Παράδειγμα

```html
<button type="button">Πατήστε εδώ</button>
```

---

## Περιεχόμενο μέσα σε ένα `<button>`

Ένα από τα σημαντικότερα πλεονεκτήματα του `<button>` είναι ότι μπορεί να περιέχει διάφορα HTML στοιχεία.

Για παράδειγμα:

```html
<button type="button"><strong>Αποθήκευση</strong></button>
```

Μπορούμε επίσης να προσθέσουμε εικόνες ή εικονίδια.

```html
<button type="button"><img src="images/save.png" alt="" width="20" height="20">Αποθήκευση</button>
```

Αυτό δεν είναι δυνατό με το στοιχείο `<input>`.

---

## Το attribute `type`

Το σημαντικότερο attribute του `<button>` είναι το `type`.

Καθορίζει ποια λειτουργία θα έχει το κουμπί.

|Τιμή|Περιγραφή|
|:-:|:-|
|`button`|Απλό κουμπί χωρίς προκαθορισμένη λειτουργία.|
|`submit`|Υποβάλλει (αποστέλλει) τη φόρμα.|
|`reset`|Επαναφέρει όλα τα πεδία της φόρμας στις αρχικές τους τιμές.|

---

### `type="button"`

Δημιουργεί ένα απλό κουμπί.

Συνήθως χρησιμοποιείται μαζί με JavaScript.

```html
<button type="button">Κάνε κλικ</button>
```

---

### `type="submit"`

Υποβάλλει τη φόρμα όταν ο χρήστης πατήσει το κουμπί.

```html
<form action="/submit" method="post">
    ...
    <button type="submit">Αποστολή</button>
</form>
```

Όταν ο χρήστης πατήσει το κουμπί, η φόρμα αποστέλλεται.

---

### `type="reset"`

Επαναφέρει όλα τα πεδία της φόρμας στις αρχικές τους τιμές.

```html
<form>
    ...
    <button type="reset">Εκκαθάριση</button>
</form>
```

Όταν ο χρήστης πατήσει το κουμπί, όλα τα πεδία της φόρμας επιστρέφουν στις αρχικές τους τιμές.

---

## `<button>` ή `<input>`

Και τα δύο μπορούν να δημιουργήσουν κουμπιά.

Ωστόσο, το `<button>` είναι πιο ευέλικτο.

|`<button>`|`<input>`|
|:-:|:-:|
|Μπορεί να περιέχει HTML στοιχεία.|Περιέχει μόνο απλό κείμενο.|
|Υποστηρίζει εικόνες και εικονίδια.|Δεν υποστηρίζει HTML περιεχόμενο.|
|Προτιμάται στις περισσότερες περιπτώσεις.|Χρησιμοποιείται κυρίως για απλά κουμπιά.|

---

## Attributes

|Attribute|Value|Περιγραφή|
|:-:|:-:|:-|
|type|button, reset, submit|Specifies the type of button|
|name|name|Specifies a name for the button|
|value|text|Specifies an initial value for the button|
|disabled|disabled|Specifies that a button should be disabled|

---

### Το attribute `disabled`

Το `disabled` απενεργοποιεί το κουμπί.

Ο χρήστης δεν μπορεί να το πατήσει.

```html
<button type="button" disabled>Μη διαθέσιμο</button>
```

---

### Τα attributes `name` και `value`

Χρησιμοποιούνται κυρίως μέσα σε φόρμες.

```html
<form action="/submit" method="post">
    <button type="submit" name="action" value="save">Αποθήκευση</button>
</form>
```

Όταν υποβληθεί η φόρμα, αποστέλλεται επίσης το ζεύγος:

```none
action=save
```

---

## Σημαντικές παρατηρήσεις

- Χρησιμοποιείτε πάντοτε το attribute `type`.
- Προτιμήστε το `<button>` αντί για το `<input>` όταν χρειάζεστε εικόνες, εικονίδια ή πιο σύνθετο περιεχόμενο.
- Το `<button>` μπορεί να χρησιμοποιηθεί τόσο μέσα όσο και έξω από μια φόρμα.
- Αν το `<button>` βρίσκεται μέσα σε μια φόρμα και δεν οριστεί το `type`, οι περισσότεροι browsers το αντιμετωπίζουν ως `type="submit"`.

> **Βέλτιστη πρακτική:** Να ορίζετε πάντοτε ρητά το `type` (`button`, `submit` ή `reset`) ώστε η συμπεριφορά του κουμπιού να είναι σαφής και προβλέψιμη.

---

## Πότε χρησιμοποιώ το `<button>` και πότε το `<input>`

Στη σύγχρονη HTML, η γενική σύσταση είναι:

> **Χρησιμοποίησε το `<button>` σχεδόν σε όλες τις περιπτώσεις που χρειάζεσαι κουμπί.**  
> Χρησιμοποίησε το `<input>` κυρίως για πεδία εισαγωγής δεδομένων (`text`, `email`, `password` κ.λπ.) ή όταν χρειάζεσαι ένα πολύ απλό κουμπί.

---

## Πότε χρησιμοποιώ το `<button>`

Το `<button>` είναι η προτιμότερη επιλογή όταν το κουμπί:

- εμφανίζει κείμενο μαζί με εικόνες ή εικονίδια,
- περιέχει HTML στοιχεία (π.χ. `<strong>`, `<span>`),
- εκτελεί ενέργειες με JavaScript,
- υποβάλλει ή επαναφέρει μια φόρμα.

### Παράδειγμα `<button>`

```html
<button type="button">
    <img src="images/save.png" alt="" width="20" height="20">
    Αποθήκευση
</button>
```

---

## Πότε χρησιμοποιώ το `<input>`

Το `<input>` χρησιμοποιείται κυρίως για **εισαγωγή δεδομένων**.

Παραδείγματα:

```html
<input type="text">
<input type="email">
<input type="password">
<input type="number">
```

Μπορεί επίσης να δημιουργήσει απλά κουμπιά:

```html
<input type="submit" value="Αποστολή">

<input type="reset" value="Εκκαθάριση">

<input type="button" value="Κλικ">
```

Ωστόσο, αυτά τα κουμπιά έχουν λιγότερες δυνατότητες από το `<button>`.

---

## Γιατί προτιμάμε το `<button>`

Το `<button>` μπορεί να περιέχει οποιοδήποτε HTML περιεχόμενο.

Παράδειγμα με μορφοποιημένο κείμενο:

```html
<button type="button">
    <strong>Αποθήκευση</strong>
</button>
```

Παράδειγμα με εικόνα:

```html
<button type="button">
    <img src="images/save.png" alt="" width="20" height="20">
    Αποθήκευση
</button>
```

Αντίθετα, το `<input>` μπορεί να εμφανίσει μόνο απλό κείμενο:

```html
<input type="button" value="Αποθήκευση">
```

---

## Συγκριτικός πίνακας

|Αν θέλω...|Χρησιμοποιώ|
|:-|:-:|
|Πεδίο κειμένου|`<input type="text">`|
|Πεδίο email|`<input type="email">`|
|Πεδίο κωδικού|`<input type="password">`|
|Checkbox|`<input type="checkbox">`|
|Radio button|`<input type="radio">`|
|Απλό κουμπί|`<button>`|
|Κουμπί με εικόνα ή εικονίδιο|`<button>`|
|Κουμπί με HTML μορφοποίηση|`<button>`|
|Υποβολή φόρμας|`<button type="submit">` *(ή `<input type="submit">`)*|
|Επαναφορά φόρμας|`<button type="reset">` *(ή `<input type="reset">`)*|

---

## Χρήση του `<button>` μέσα και έξω από φόρμες

Το στοιχείο `<button>` μπορεί να χρησιμοποιηθεί **τόσο μέσα όσο και έξω από μια φόρμα (`<form>`)**.

Η συμπεριφορά του εξαρτάται από την τιμή του attribute `type` και από το αν το κουμπί βρίσκεται μέσα σε μια φόρμα.

---

### Το `type="button"`

Το `type="button"` δημιουργεί ένα **απλό κουμπί**.

Δεν εκτελεί κάποια προκαθορισμένη ενέργεια από μόνο του και συνήθως χρησιμοποιείται μαζί με JavaScript.

#### Έξω από φόρμα `type="button"`

```html
<button type="button">
    Άνοιγμα μενού
</button>
```

Το κουμπί είναι απολύτως λειτουργικό και μπορεί να χρησιμοποιηθεί για:

- άνοιγμα ενός μενού,
- εμφάνιση ενός παραθύρου (modal),
- αλλαγή θέματος (Dark/Light Mode),
- εμφάνιση ή απόκρυψη περιεχομένου,
- οποιαδήποτε άλλη ενέργεια μέσω JavaScript.

---

#### Μέσα σε φόρμα `type="button"`

```html
<form>
    <input type="text" name="username">
    <button type="button">
        Έλεγχος
    </button>
</form>
```

Παρότι βρίσκεται μέσα σε φόρμα, το κουμπί **δεν υποβάλλει** τη φόρμα.

Χρησιμοποιείται όταν θέλουμε να εκτελέσουμε κάποια άλλη ενέργεια χωρίς να γίνει αποστολή των δεδομένων.

---

### Το `type="submit"`

Το `type="submit"` χρησιμοποιείται για την **υποβολή μιας φόρμας**.

#### Μέσα σε φόρμα `type="submit"`

```html
<form action="/submit" method="post">
    <input type="text" name="username">
    <button type="submit">
        Αποστολή
    </button>
</form>
```

Όταν ο χρήστης πατήσει το κουμπί, η φόρμα αποστέλλεται.

---

#### Έξω από φόρμα `type="submit"`

```html
<button type="submit">
    Αποστολή
</button>
```

Η σύνταξη είναι απολύτως έγκυρη.

Ωστόσο, επειδή δεν υπάρχει φόρμα, **δεν υπάρχει τίποτα να υποβληθεί**.

Το κουμπί δεν εκτελεί κάποια ιδιαίτερη ενέργεια.

> **Συμπέρασμα:** Το `type="submit"` έχει νόημα μόνο όταν υπάρχει φόρμα.

---

### Το `type="reset"`

Το `type="reset"` επαναφέρει όλα τα πεδία μιας φόρμας στις αρχικές τους τιμές.

#### Μέσα σε φόρμα `type="reset"`

```html
<form>
    <input type="text" name="username">
    <button type="reset">
        Εκκαθάριση
    </button>
</form>
```

Όταν ο χρήστης πατήσει το κουμπί, όλα τα πεδία της φόρμας επανέρχονται στην αρχική τους κατάσταση.

---

#### Έξω από φόρμα `type="reset"`

```html
<button type="reset">
    Εκκαθάριση
</button>
```

Η σύνταξη είναι επίσης έγκυρη.

Επειδή όμως δεν υπάρχει φόρμα, **δεν υπάρχουν πεδία για επαναφορά** και το κουμπί δεν έχει κάποια πρακτική λειτουργία.

> **Συμπέρασμα:** Το `type="reset"` χρησιμοποιείται μόνο μέσα σε φόρμες.

---

### Αν δεν ορίσουμε το `type`

Αν ένα `<button>` βρίσκεται μέσα σε μια φόρμα και **δεν ορίσουμε το attribute `type`**, οι περισσότεροι browsers το αντιμετωπίζουν ως:

```html
<button type="submit">
```

Παράδειγμα:

```html
<form>
    <input type="text">
    <button>
        Αποστολή
    </button>
</form>
```

Πατώντας το κουμπί, η φόρμα θα υποβληθεί.

Για τον λόγο αυτό, θεωρείται **βέλτιστη πρακτική** να ορίζουμε πάντοτε ρητά το `type`.

---

## Πότε χρησιμοποιώ κάθε τύπο

|Υπάρχει φόρμα;|`button`|`submit`|`reset`|
|:-:|:-:|:-:|:-:|
|❌ Όχι|✔ Απλό κουμπί|✔ Έγκυρο αλλά χωρίς πρακτική λειτουργία|✔ Έγκυρο αλλά χωρίς πρακτική λειτουργία|
|✅ Ναι|✔ Απλό κουμπί|✔ Υποβάλλει τη φόρμα|✔ Επαναφέρει τη φόρμα|

---

## Βέλτιστη πρακτική

- Χωρίς φόρμα, χρησιμοποιείτε πάντοτε:

```html
<button type="button">
```

- Μέσα σε φόρμα:
  - `type="submit"` για υποβολή της φόρμας.
  - `type="reset"` για επαναφορά των πεδίων.
  - `type="button"` για οποιαδήποτε άλλη ενέργεια.

> **Κανόνας που αξίζει να θυμάστε:**  
> Το `submit` και το `reset` αποκτούν νόημα μόνο όταν σχετίζονται με μια φόρμα. Σε όλες τις υπόλοιπες περιπτώσεις, προτιμήστε το `type="button"`.

---

## Σύγχρονη πρακτική

Σήμερα οι περισσότεροι προγραμματιστές χρησιμοποιούν:

- **`<input>`** για την εισαγωγή δεδομένων.
- **`<button>`** για όλα τα κουμπιά.

Έτσι ο κώδικας είναι πιο ευέλικτος, πιο εύκολος στη μορφοποίηση και πιο συμβατός με σύγχρονα CSS frameworks.

> **Κανόνας που αξίζει να θυμάσαι:**  
> **`<input>` για εισαγωγή δεδομένων.**  
> **`<button>` για κουμπιά και ενέργειες.**
