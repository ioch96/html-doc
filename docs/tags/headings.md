# HTML `<h1>` έως `<h6>` Tags

Τα tags `<h1>` έως `<h6>` χρησιμοποιούνται για τον ορισμό **επικεφαλίδων (headings)** σε ένα έγγραφο HTML.

Το `<h1>` ορίζει την **σημαντικότερη επικεφαλίδα**, ενώ το `<h6>` τη **λιγότερο σημαντική**.

Οι επικεφαλίδες βοηθούν:

- Στην οργάνωση του περιεχομένου της σελίδας.
- Στη βελτίωση της αναγνωσιμότητας.
- Στην προσβασιμότητα (Accessibility).
- Στη βελτιστοποίηση για τις μηχανές αναζήτησης (SEO).

> **Σημείωση:** Συνιστάται να χρησιμοποιείτε **μόνο ένα** στοιχείο `<h1>` σε κάθε σελίδα. Αυτό θα πρέπει να αντιπροσωπεύει τον κύριο τίτλο ή το βασικό θέμα της σελίδας.

## Παράδειγμα

```html
<!DOCTYPE html>
<html lang="el">
  <head>
    <title>Τίτλος του Εγγράφου</title>
  </head>
  <body>
    <h1>Αυτή είναι η επικεφαλίδα 1</h1>
    <h2>Αυτή είναι η επικεφαλίδα 2</h2>
    <h3>Αυτή είναι η επικεφαλίδα 3</h3>
    <h4>Αυτή είναι η επικεφαλίδα 4</h4>
    <h5>Αυτή είναι η επικεφαλίδα 5</h5>
    <h6>Αυτή είναι η επικεφαλίδα 6</h6>
  </body>
</html>
```

## Ιεραρχία των επικεφαλίδων

|Tag|Περιγραφή|
|:---:|-----------|
|`<h1>`|Κύρια επικεφαλίδα της σελίδας.|
|`<h2>`|Κύρια ενότητα της σελίδας.|
|`<h3>`|Υποενότητα μιας `<h2>`.|
|`<h4>`|Υποενότητα μιας `<h3>`.|
|`<h5>`|Χρησιμοποιείται για βαθύτερα επίπεδα οργάνωσης.|
|`<h6>`|Το χαμηλότερο επίπεδο επικεφαλίδας.|

> **Σημείωση:** Μην παραλείπετε επίπεδα επικεφαλίδων. Ξεκινήστε με `<h1>`, στη συνέχεια χρησιμοποιήστε `<h2>`, μετά `<h3>` κ.ο.κ., ώστε να διατηρείται μια σωστή ιεραρχική δομή.

## Παράδειγμα σωστής δομής

```html
<h1>HTML Tutorial</h1>

<h2>Εισαγωγή</h2>
<p>...</p>

<h2>HTML Elements</h2>

<h3>Headings</h3>
<p>...</p>

<h3>Paragraphs</h3>
<p>...</p>

<h2>Συμπέρασμα</h2>
<p>...</p>
```

## Best Practices

- Χρησιμοποιείτε **μόνο ένα** `<h1>` ανά σελίδα.
- Ακολουθείτε πάντα τη σωστή ιεραρχία (`<h1>` &rarr; `<h2>` &rarr; `<h3>` &rarr; ...).
- Μην επιλέγετε κάποιο tag επικεφαλίδας μόνο επειδή έχει μεγαλύτερο ή μικρότερο μέγεθος γραμματοσειράς. Αν θέλετε να αλλάξετε την εμφάνιση μιας επικεφαλίδας, χρησιμοποιήστε **CSS**.
- Οι επικεφαλίδες πρέπει να περιγράφουν με σαφήνεια το περιεχόμενο της ενότητας που ακολουθεί.

> **Σημείωση:** Τα tags `<h1>` έως `<h6>` έχουν **σημασιολογική αξία (semantic meaning)**. Οι browsers, οι μηχανές αναζήτησης και οι τεχνολογίες υποβοήθησης (όπως οι screen readers) χρησιμοποιούν την ιεραρχία των επικεφαλίδων για να κατανοήσουν τη δομή και το περιεχόμενο της ιστοσελίδας.
