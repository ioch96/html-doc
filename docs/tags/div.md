# HTML `<div>` Tag

Το tag `<div>` χρησιμοποιείται για να ορίσει μια **ενότητα (division)** μέσα σε ένα έγγραφο HTML.

Το `<div>` λειτουργεί ως ένα **container** που χρησιμοποιείται για την ομαδοποίηση άλλων HTML στοιχείων.

Στη συνέχεια, το περιεχόμενό του μπορεί:

- Να μορφοποιηθεί με **CSS**.
- Να τροποποιηθεί δυναμικά με **JavaScript**.
- Να οργανωθεί καλύτερα μέσα στη σελίδα.

Το `<div>` είναι από τα πιο συχνά χρησιμοποιούμενα στοιχεία στην HTML και αποτελεί βασικό δομικό στοιχείο στις περισσότερες ιστοσελίδες και εφαρμογές.

## Παράδειγμα

```html
<!DOCTYPE html>
<html lang="el">
<head>
  <title>Τίτλος του Εγγράφου</title>
</head>

<body>

  <div>
    <h1>Καλώς ήρθατε</h1>
    <p>Αυτό είναι ένα απλό παράδειγμα χρήσης του στοιχείου div.</p>
  </div>

</body>
</html>
```

## Το `<div>` ως Container

Μπορείτε να τοποθετήσετε **οποιοδήποτε HTML στοιχείο** μέσα σε ένα `<div>`.

Παράδειγμα:

```html
<div>
  <h2>Προϊόν</h2>

  <img src="laptop.jpg" alt="Laptop">

  <p>Ισχυρός φορητός υπολογιστής.</p>

  <button>Αγορά</button>
</div>
```

## Χρήση με CSS

Το `<div>` χρησιμοποιείται συχνά μαζί με τα attributes `class` ή `id`.

```html
<div class="card">
  <h2>HTML</h2>
  <p>Η γλώσσα σήμανσης του Web.</p>
</div>
```

```css
.card {
  background-color: #f4f4f4;
  padding: 20px;
  border-radius: 8px;
}
```

## Χρήση με JavaScript

Το `<div>` μπορεί να επιλεγεί και να τροποποιηθεί εύκολα μέσω JavaScript.

```html
<div id="message">
  Γεια σου!
</div>
```

```javascript
document.getElementById("message").textContent = "Καλώς ήρθες!";
```

## Χαρακτηριστικά του `<div>`

- Είναι **block-level element**.
- Καταλαμβάνει όλο το διαθέσιμο πλάτος του γονικού στοιχείου.
- Ο browser τοποθετεί αυτόματα αλλαγή γραμμής πριν και μετά από αυτό.
- Δεν έχει συγκεκριμένη σημασιολογία (semantic meaning).

## Διαφορά μεταξύ `<div>` και `<span>`

|`<div>`|`<span>`|
|----------|----------|
|Είναι **block-level** στοιχείο.|Είναι **inline** στοιχείο.|
|Ξεκινά πάντα σε νέα γραμμή.|Παραμένει στην ίδια γραμμή με το υπόλοιπο κείμενο.|
|Χρησιμοποιείται για ομαδοποίηση μεγαλύτερων τμημάτων περιεχομένου.|Χρησιμοποιείται για μικρά τμήματα κειμένου ή inline στοιχείων.|
|Χρησιμοποιείται κυρίως για layout και containers.|Χρησιμοποιείται κυρίως για μορφοποίηση ή επιλογή μικρών τμημάτων κειμένου.|

## Βέλτιστες Πρακτικές (Best Practices)

- Χρησιμοποιείτε το `<div>` όταν χρειάζεστε ένα **γενικό container** χωρίς ιδιαίτερη σημασιολογική σημασία.
- Αν υπάρχει πιο κατάλληλο σημασιολογικό στοιχείο, όπως `<header>`, `<main>`, `<section>`, `<article>` ή `<footer>`, προτιμήστε αυτό αντί για `<div>`.
- Χρησιμοποιείτε `class` ή `id` για να μορφοποιείτε ή να επιλέγετε το `<div>` με CSS και JavaScript.

> **Σημείωση:** Το `<div>` είναι ίσως το πιο συχνά χρησιμοποιούμενο HTML στοιχείο. Αν και δεν έχει σημασιολογική αξία, αποτελεί βασικό εργαλείο για την οργάνωση του περιεχομένου.
