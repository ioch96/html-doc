# HTML `<th>` Tag

Το tag `<th>` χρησιμοποιείται για τη δημιουργία ενός **κελιού επικεφαλίδας (header cell)** σε έναν πίνακα HTML.

Ένα κελί επικεφαλίδας χρησιμοποιείται για να **περιγράφει τα δεδομένα** μιας στήλης ή μιας γραμμής του πίνακα.

Συνήθως βρίσκεται στην πρώτη γραμμή του πίνακα, αλλά μπορεί επίσης να χρησιμοποιηθεί και στην πρώτη στήλη.

---

## Τα δύο είδη κελιών σε έναν πίνακα

Ένας πίνακας HTML αποτελείται από δύο βασικά είδη κελιών:

- **Κελιά επικεφαλίδων (Header cells)**, τα οποία δημιουργούνται με το tag `<th>`.
- **Κελιά δεδομένων (Data cells)**, τα οποία δημιουργούνται με το tag `<td>`.

Τα `<th>` περιγράφουν το νόημα των δεδομένων, ενώ τα `<td>` περιέχουν τα ίδια τα δεδομένα.

---

## Παράδειγμα

```html
<table>
    <tr>
        <th>Όνομα</th>
        <th>Ηλικία</th>
        <th>Πόλη</th>
    </tr>
    <tr>
        <td>Μαρία</td>
        <td>25</td>
        <td>Αθήνα</td>
    </tr>
    <tr>
        <td>Γιώργος</td>
        <td>31</td>
        <td>Θεσσαλονίκη</td>
    </tr>
</table>
```

---

## Τι κάνει το `<th>`

Το `<th>` δημιουργεί ένα κελί επικεφαλίδας.

Η λειτουργία του είναι να περιγράφει τι αντιπροσωπεύουν τα δεδομένα μιας στήλης ή μιας γραμμής.

Για παράδειγμα:

```html
<th>Όνομα</th>
```

δηλώνει ότι τα κελιά που βρίσκονται κάτω από αυτό περιέχουν ονόματα.

---

## Προεπιλεγμένη εμφάνιση

Από προεπιλογή, οι περισσότεροι browsers εμφανίζουν το περιεχόμενο ενός `<th>`:

- με έντονη γραφή (bold),
- κεντραρισμένο.

> Η εμφάνιση αυτή είναι η προεπιλεγμένη συμπεριφορά του browser και μπορεί να αλλάξει με CSS.

---

## Γιατί χρησιμοποιούμε το `<th>`

Το `<th>` δεν χρησιμοποιείται μόνο για να κάνει το κείμενο έντονο.

Ο βασικός σκοπός του είναι να δηλώσει ότι το συγκεκριμένο κελί αποτελεί **επικεφαλίδα**.

Αυτό βοηθά:

- τον browser να κατανοήσει καλύτερα τη δομή του πίνακα,
- τις μηχανές αναζήτησης να ερμηνεύσουν σωστά τα δεδομένα,
- τα βοηθητικά προγράμματα (screen readers) να ανακοινώνουν σωστά τις επικεφαλίδες στους χρήστες.

Γι' αυτό προτιμάμε πάντα το `<th>` αντί για ένα `<td>` όταν το κελί λειτουργεί ως επικεφαλίδα.

---

## Attributes

|Attribute|Value|Περιγραφή|
|:-:|:-:|:-|
|`colspan`|number|Καθορίζει πόσες στήλες θα καταλαμβάνει το κελί επικεφαλίδας.|
|`rowspan`|number|Καθορίζει πόσες γραμμές θα καταλαμβάνει το κελί επικεφαλίδας.|
|`scope`|`col`, `row`|Δηλώνει αν το κελί είναι επικεφαλίδα στήλης ή γραμμής.|

---

### Το attribute `colspan`

Το `colspan` επιτρέπει σε ένα κελί επικεφαλίδας να εκτείνεται σε περισσότερες από μία στήλες.

#### Παράδειγμα `colspan`

```html
<table>
    <tr>
        <th colspan="2">Στοιχεία Μαθητή</th>
    </tr>
    <tr>
        <th>Όνομα</th>
        <th>Ηλικία</th>
    </tr>
    <tr>
        <td>Μαρία</td>
        <td>25</td>
    </tr>
</table>
```

Το πρώτο κελί επικεφαλίδας καταλαμβάνει δύο στήλες.

---

### Το attribute `rowspan`

Το `rowspan` επιτρέπει σε ένα κελί επικεφαλίδας να εκτείνεται σε περισσότερες από μία γραμμές.

#### Παράδειγμα `rowspan`

```html
<table>
    <tr>
        <th rowspan="2">Όνομα</th>
        <th>Μάθημα</th>
    </tr>
    <tr>
        <th>Βαθμός</th>
    </tr>
    <tr>
        <td>Μαρία</td>
        <td>18</td>
    </tr>
</table>
```

Το πρώτο κελί επικεφαλίδας καταλαμβάνει δύο διαδοχικές γραμμές.

---

### Το attribute `scope`

Το attribute `scope` χρησιμοποιείται για να δηλώσει σε ποια δεδομένα αναφέρεται ένα κελί επικεφαλίδας.

Οι πιο συνηθισμένες τιμές είναι:

- `col` → επικεφαλίδα στήλης
- `row` → επικεφαλίδα γραμμής

#### Παράδειγμα (`scope="col"`)

```html
<table>
    <tr>
        <th scope="col">Όνομα</th>
        <th scope="col">Ηλικία</th>
    </tr>
    <tr>
        <td>Μαρία</td>
        <td>25</td>
    </tr>
</table>
```

Κάθε `<th>` είναι επικεφαλίδα της αντίστοιχης στήλης.

#### Παράδειγμα (`scope="row"`)

```html
<table>
    <tr>
        <th scope="row">Μαρία</th>
        <td>25</td>
    </tr>
    <tr>
        <th scope="row">Γιώργος</th>
        <td>31</td>
    </tr>
</table>
```

Εδώ, το πρώτο κελί κάθε γραμμής λειτουργεί ως επικεφαλίδα της συγκεκριμένης γραμμής.

> Το `scope` δεν αλλάζει την εμφάνιση του πίνακα. Χρησιμοποιείται κυρίως για σημασιολογία και προσβασιμότητα.

---

## Καθορισμός πλάτους ενός `<th>`

Σε παλαιότερες εκδόσεις της HTML μπορούσαμε να χρησιμοποιήσουμε το attribute `width` απευθείας μέσα στο `<th>`.

```html
<table>
    <tr>
        <th width="100">Header 1</th>
        <th width="200">Header 2</th>
        <th>Header 3</th>
    </tr>

    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
        <td>Data 3</td>
    </tr>
</table>
```

Στο παραπάνω παράδειγμα:

- το πρώτο `<th>` έχει πλάτος `100` pixels,
- το δεύτερο `<th>` έχει πλάτος `200` pixels,
- το τρίτο `<th>` δεν έχει καθορισμένο πλάτος.

Ωστόσο, το attribute `width` θεωρείται **παρωχημένο (obsolete)** στην HTML5 και δεν συνιστάται να χρησιμοποιείται σε σύγχρονες ιστοσελίδες.

---

### Προτεινόμενος τρόπος με CSS

Το πλάτος ενός κελιού επικεφαλίδας πρέπει να καθορίζεται με την ιδιότητα CSS `width`.

```html
<table>
    <tr>
        <th style="width: 100px;">Header 1</th>
        <th style="width: 200px;">Header 2</th>
        <th>Header 3</th>
    </tr>

    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
        <td>Data 3</td>
    </tr>
</table>
```

Η ιδιότητα:

```css
width: 100px;
```

δηλώνει ότι το κελί πρέπει να έχει πλάτος `100` pixels.

---

### Χρήση CSS κλάσεων

Σε ένα πραγματικό project είναι προτιμότερο να αποφεύγουμε το inline CSS και να χρησιμοποιούμε κλάσεις.

```html
<style>
    .small-column {
        width: 100px;
    }

    .large-column {
        width: 200px;
    }
</style>

<table>
    <tr>
        <th class="small-column">Header 1</th>
        <th class="large-column">Header 2</th>
        <th>Header 3</th>
    </tr>

    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
        <td>Data 3</td>
    </tr>
</table>
```

Με αυτόν τον τρόπο, η HTML περιγράφει τη δομή του πίνακα, ενώ το CSS ελέγχει την εμφάνισή του.

---

### Πλάτος σε ποσοστό

Μπορούμε επίσης να χρησιμοποιήσουμε ποσοστά.

```html
<table style="width: 100%;">
    <tr>
        <th style="width: 25%;">Προϊόν</th>
        <th style="width: 50%;">Περιγραφή</th>
        <th style="width: 25%;">Τιμή</th>
    </tr>

    <tr>
        <td>Ακουστικά</td>
        <td>Ασύρματα ακουστικά με Bluetooth.</td>
        <td>49,90 €</td>
    </tr>
</table>
```

Εδώ:

- η πρώτη στήλη καταλαμβάνει το `25%` του πίνακα,
- η δεύτερη στήλη καταλαμβάνει το `50%`,
- η τρίτη στήλη καταλαμβάνει το `25%`.

> **Σημείωση:** Η ιδιότητα `width` επηρεάζει ουσιαστικά ολόκληρη τη στήλη και όχι μόνο το συγκεκριμένο κελί επικεφαλίδας. Το τελικό πλάτος μπορεί επίσης να επηρεάζεται από το περιεχόμενο των κελιών και από τους υπόλοιπους κανόνες CSS.
>
> **Βέλτιστη πρακτική:** Αποφύγετε το παλιό attribute `width` μέσα στο `<th>`. Χρησιμοποιείτε την ιδιότητα CSS `width`, κατά προτίμηση μέσω εξωτερικού αρχείου CSS ή CSS κλάσεων.

---

## Σημαντικές παρατηρήσεις

- Το `<th>` πρέπει πάντα να βρίσκεται μέσα σε ένα `<tr>`.
- Συνήθως χρησιμοποιείται στην πρώτη γραμμή ή στην πρώτη στήλη του πίνακα.
- Μπορεί να χρησιμοποιηθεί μαζί με τα `colspan` και `rowspan` για τη δημιουργία πιο σύνθετων επικεφαλίδων.
- Παρόλο που οι browsers εμφανίζουν το περιεχόμενό του με έντονη γραφή, αυτός **δεν είναι ο λόγος ύπαρξής του**. Ο πραγματικός σκοπός του είναι να δηλώνει ότι το κελί αποτελεί επικεφαλίδα.

---
