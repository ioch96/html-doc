# HTML `<sup>` Tag

Το tag `<sup>` χρησιμοποιείται για την εμφάνιση **εκθέτη (superscript)**. Το περιεχόμενό του εμφανίζεται ελαφρώς **πάνω από τη βασική γραμμή του κειμένου** και συνήθως με μικρότερο μέγεθος γραμματοσειράς.

Το στοιχείο `<sup>` χρησιμοποιείται κυρίως για:

- μαθηματικούς εκθέτες,
- μονάδες μέτρησης,
- υποσημειώσεις (footnotes),
- τακτικούς αριθμούς σε ορισμένες γλώσσες.

## Παράδειγμα 1

Χρήση εκθέτη σε μαθηματική έκφραση.

```html
<p>x<sup>2</sup> + y<sup>2</sup> = z<sup>2</sup></p>
```

---

## Παράδειγμα 2

Χρήση σε μονάδες μέτρησης.

```html
<p>Η επιφάνεια του δωματίου είναι 25 m<sup>2</sup>.</p>
```

---

## Παράδειγμα 3

Χρήση σε κυβικές μονάδες.

```html
<p>Ο όγκος του δοχείου είναι 3 m<sup>3</sup>.</p>
```

---

## Παράδειγμα 4

Χρήση σε υποσημειώσεις.

```html
<p>
    Η HTML είναι η βασική γλώσσα δημιουργίας ιστοσελίδων.<sup>1</sup>
</p>
```
