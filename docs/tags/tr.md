# HTML `<tr>` Tag

Το tag `<tr>` χρησιμοποιείται για τη δημιουργία μιας **γραμμής (row)** σε έναν πίνακα HTML.

Κάθε γραμμή ενός πίνακα αποτελείται από ένα στοιχείο `<tr>` και περιέχει ένα ή περισσότερα κελιά.

Τα κελιά αυτά μπορούν να είναι:

- κελιά επικεφαλίδων (`<th>`),
- κελιά δεδομένων (`<td>`).

---

## Παράδειγμα

```html
<table>
    <tr>
        <th>Όνομα</th>
        <th>Ηλικία</th>
    </tr>
    <tr>
        <td>Μαρία</td>
        <td>25</td>
    </tr>
    <tr>
        <td>Γιώργος</td>
        <td>31</td>
    </tr>
</table>
```

---

## Τι κάνει το `<tr>`

Το `<tr>` δημιουργεί μία **οριζόντια γραμμή** μέσα στον πίνακα.

Κάθε νέα γραμμή αντιστοιχεί σε ένα νέο στοιχείο `<tr>`.

Για παράδειγμα:

```html
<tr>
    <td>Μαρία</td>
    <td>25</td>
</tr>
```

δημιουργεί μία γραμμή με δύο κελιά.

Αν προσθέσουμε ακόμη ένα `<tr>`:

```html
<tr>
    <td>Γιώργος</td>
    <td>31</td>
</tr>
```

δημιουργείται μια δεύτερη γραμμή στον ίδιο πίνακα.

---

## Τι μπορεί να περιέχει ένα `<tr>`

Ένα στοιχείο `<tr>` μπορεί να περιέχει:

- ένα ή περισσότερα στοιχεία `<td>`,
- ένα ή περισσότερα στοιχεία `<th>`,

Παράδειγμα:

```html
<tr>
    <th>Όνομα</th>
    <th>Ηλικία</th>
</tr>
```

ή

```html
<tr>
    <td>Μαρία</td>
    <td>25</td>
</tr>
```

---

## Το `<tr>` δεν εμφανίζει περιεχόμενο από μόνο του

Το `<tr>` δημιουργεί μόνο τη γραμμή του πίνακα.

Δεν εμφανίζει κανένα περιεχόμενο αν δεν περιέχει κελιά.

Για παράδειγμα:

```html
<tr></tr>
```

δημιουργεί μια κενή γραμμή.

Συνήθως, κάθε `<tr>` περιέχει τουλάχιστον ένα `<th>` ή `<td>`.

---

## Θέση του `<tr>` μέσα στον πίνακα

Το `<tr>` πρέπει πάντα να βρίσκεται μέσα σε ένα στοιχείο `<table>`.

Μπορεί να είναι:

- απευθείας μέσα στο `<table>`,
- μέσα σε ένα `<thead>`,
- μέσα σε ένα `<tbody>`,
- μέσα σε ένα `<tfoot>`.

Τα στοιχεία `<thead>`, `<tbody>` και `<tfoot>` θα αναλυθούν σε ξεχωριστές ενότητες.

---

## Παράδειγμα με πολλές γραμμές

```html
<table>
    <tr>
        <th>Προϊόν</th>
        <th>Τιμή</th>
    </tr>
    <tr>
        <td>Laptop</td>
        <td>900€</td>
    </tr>
    <tr>
        <td>Οθόνη</td>
        <td>250€</td>
    </tr>
    <tr>
        <td>Πληκτρολόγιο</td>
        <td>60€</td>
    </tr>
</table>
```

Σε αυτό το παράδειγμα:

- το πρώτο `<tr>` περιέχει τις επικεφαλίδες,
- τα επόμενα τρία `<tr>` περιέχουν τα δεδομένα του πίνακα.

---

## Υποχρεωτικό tag κλεισίματος

Το `<tr>` πρέπει να κλείνει με το αντίστοιχο tag `</tr>`.

Σωστό:

```html
<tr>
    <td>Μαρία</td>
</tr>
```

Λάθος:

```html
<tr>
    <td>Μαρία</td>
```

---

## Browser Support

Το tag `<tr>` υποστηρίζεται από όλους τους σύγχρονους browsers.

- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari
- Opera
