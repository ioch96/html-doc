# HTML `<p>` Tag

Το tag `<p>` χρησιμοποιείται για να ορίσει μια **παράγραφο (paragraph)**.

Κάθε παράγραφος αποτελεί μια ξεχωριστή ενότητα κειμένου. Οι περισσότεροι browsers προσθέτουν αυτόματα **ένα κατακόρυφο κενό (margin)** πριν και μετά από κάθε στοιχείο `<p>`.

> **Σημείωση:** Χρησιμοποιήστε **CSS** για να μορφοποιήσετε τις παραγράφους (π.χ. στοίχιση, μέγεθος γραμματοσειράς, χρώμα, αποστάσεις κ.λπ.).

## Παράδειγμα 1

```html
<!DOCTYPE html>
<html lang="el">
  <head>
    <title>Τίτλος του Εγγράφου</title>
  </head>

  <body>
    <h1>Το στοιχείο p</h1>

    <p>Αυτή είναι μια παράγραφος.</p>
    <p>Αυτή είναι μια ακόμη παράγραφος.</p>
    <p>Και αυτή είναι μια τρίτη παράγραφος.</p>
  </body>
</html>
```

## Πώς χειρίζεται ο browser τα κενά και τις αλλαγές γραμμής

Παρόλο που το tag `<p>` χρησιμοποιείται για τη δημιουργία παραγράφων, ο browser **αγνοεί** τις επιπλέον αλλαγές γραμμής και τα κενά που υπάρχουν στον πηγαίο κώδικα.

Η HTML μετατρέπει:

- Τις αλλαγές γραμμής (line breaks)
- Τα tab
- Τα πολλαπλά κενά (spaces)

σε **ένα μόνο κενό** κατά την εμφάνιση του κειμένου.

## Παράδειγμα 2

```html
<!DOCTYPE html>
<html lang="el">
  <head>
    <title>Τίτλος του Εγγράφου</title>
  </head>

  <body>

    <p>
      Η Μπόνι μου βρίσκεται πέρα από τον ωκεανό.
      Η Μπόνι μου βρίσκεται πέρα από τη θάλασσα.
      Η Μπόνι μου βρίσκεται πέρα από τον ωκεανό.
      Αχ, φέρε πίσω τη Μπόνι μου.
    </p>

    <p>
      Παρατηρήστε ότι ο browser αγνοεί τις αλλαγές γραμμής
      που υπάρχουν στον πηγαίο κώδικα.
    </p>

  </body>
</html>
```

## Σημαντικό

Ο παρακάτω κώδικας:

```html
<p>
Γραμμή 1
Γραμμή 2
Γραμμή 3
</p>
```

εμφανίζεται στον browser ως:

```text
Γραμμή 1 Γραμμή 2 Γραμμή 3
```

και **όχι** ως:

```text
Γραμμή 1
Γραμμή 2
Γραμμή 3
```

Αν θέλετε να εμφανίζονται οι αλλαγές γραμμής, μπορείτε να χρησιμοποιήσετε:

- Το tag `<br>` για αλλαγές γραμμής.
- Το tag `<pre>` για να διατηρήσετε την ακριβή μορφοποίηση του κειμένου.
