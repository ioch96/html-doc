# HTML `<tbody>` Tag

Το tag `<tbody>` χρησιμοποιείται για να **ομαδοποιεί το κύριο σώμα (body)** ενός πίνακα HTML.

Το σώμα του πίνακα περιέχει συνήθως τα πραγματικά δεδομένα του πίνακα, δηλαδή τις γραμμές με τα στοιχεία που θέλουμε να εμφανίσουμε.

Το `<tbody>` λειτουργεί ως **container** για όλες τις γραμμές δεδομένων του πίνακα.

---

## Σύνταξη

```html
<tbody>
    ...
</tbody>
```

---

## Παράδειγμα

```html
<table>
    <thead>
        <tr>
            <th>Όνομα</th>
            <th>Ηλικία</th>
            <th>Πόλη</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Μαρία</td>
            <td>25</td>
            <td>Αθήνα</td>
        </tr>
        <tr>
            <td>Γιώργος</td>
            <td>31</td>
            <td>Θεσσαλονίκη</td>
        </tr>
    </tbody>
</table>
```

---

## Τι κάνει το `<tbody>`

Το `<tbody>` ομαδοποιεί όλες τις γραμμές που περιέχουν τα δεδομένα του πίνακα.

Συνήθως περιέχει μία ή περισσότερες γραμμές (`<tr>`), μέσα στις οποίες υπάρχουν τα κελιά δεδομένων (`<td>`).

Παράδειγμα:

```html
<tbody>
    <tr>
        <td>Μαρία</td>
        <td>25</td>
    </tr>
    <tr>
        <td>Γιώργος</td>
        <td>31</td>
    </tr>
</tbody>
```

Όλες οι γραμμές μέσα στο `<tbody>` θεωρούνται μέρος του κύριου περιεχομένου του πίνακα.

---

## Το `<tbody>` πρέπει να περιέχει `<tr>`

Το `<tbody>` δεν μπορεί να περιέχει απευθείας στοιχεία `<td>` ή `<th>`.

Πρέπει να περιέχει ένα ή περισσότερα στοιχεία `<tr>`.

Σωστό:

```html
<tbody>
    <tr>
        <td>Μαρία</td>
        <td>25</td>
    </tr>
</tbody>
```

Λάθος:

```html
<tbody>
    <td>Μαρία</td>
    <td>25</td>
</tbody>
```

Τα στοιχεία `<td>` και `<th>` πρέπει πάντα να βρίσκονται μέσα σε ένα `<tr>`.

---

## Το `<tbody>` χρησιμοποιείται μαζί με τα `<thead>` και `<tfoot>`

Ένας πίνακας μπορεί να χωριστεί σε τρία βασικά τμήματα:

- `<thead>` &rarr; κεφαλίδα του πίνακα.
- `<tbody>` &rarr; κύριο σώμα του πίνακα.
- `<tfoot>` &rarr; υποσέλιδο του πίνακα.

Παράδειγμα:

```html
<table>
    <thead>
        ...
    </thead>
    <tbody>
        ...
    </tbody>
    <tfoot>
        ...
    </tfoot>
</table>
```

Η διάκριση αυτή κάνει τον πίνακα πιο οργανωμένο και πιο ευανάγνωστο.

---

## Είναι υποχρεωτικό το `<tbody>`

Όχι.

Ένας απλός πίνακας μπορεί να λειτουργήσει και χωρίς να γράψουμε το `<tbody>`.

Παράδειγμα:

```html
<table>
    <tr>
        <td>Μαρία</td>
    </tr>
</table>
```

Οι περισσότεροι σύγχρονοι browsers δημιουργούν αυτόματα ένα `<tbody>` στο DOM, ακόμη κι αν δεν υπάρχει στον HTML κώδικα.

Ωστόσο, θεωρείται **καλή πρακτική** να το γράφουμε ρητά, ειδικά σε μεγαλύτερους πίνακες, γιατί κάνει τον κώδικα πιο σαφή και πιο οργανωμένο.

---

## Επηρεάζει την εμφάνιση του πίνακα

Όχι.

Το `<tbody>` **δεν αλλάζει από μόνο του την εμφάνιση** του πίνακα.

Η μορφοποίηση των γραμμών και των κελιών γίνεται συνήθως με CSS.

---

## Γιατί χρησιμοποιούμε το `<tbody>`

Η χρήση του `<tbody>` προσφέρει αρκετά πλεονεκτήματα.

### Καλύτερη οργάνωση

Διαχωρίζει καθαρά:

- την κεφαλίδα,
- τα δεδομένα,
- το υποσέλιδο.

---

### Καλύτερη σημασιολογία

Δηλώνει ότι το συγκεκριμένο τμήμα περιέχει τα κύρια δεδομένα του πίνακα.

Αυτό βοηθά:

- τους browsers,
- τις μηχανές αναζήτησης,
- τα βοηθητικά προγράμματα προσβασιμότητας (screen readers).

---

### Μεγάλοι πίνακες

Σε μεγάλους πίνακες, οι browsers ή άλλα προγράμματα μπορούν να αξιοποιήσουν το `<tbody>` ώστε:

- να διαχειρίζονται καλύτερα την κύλιση του πίνακα,
- να διαχωρίζουν τα δεδομένα από την κεφαλίδα και το υποσέλιδο,
- να βελτιώνουν την εκτύπωση πολύ μεγάλων πινάκων.

Η συμπεριφορά αυτή εξαρτάται από τον browser ή από το CSS που χρησιμοποιείται.

---

## Υποχρεωτικό tag κλεισίματος

Το `<tbody>` πρέπει να κλείνει με το αντίστοιχο tag.

Σωστό:

```html
<tbody>

</tbody>
```

Λάθος:

```html
<tbody>
```

---

## Browser Support

Το tag `<tbody>` υποστηρίζεται από όλους τους σύγχρονους browsers.

- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari
- Opera
