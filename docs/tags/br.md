# HTML `<br>` Tag

Το tag `<br>` χρησιμοποιείται για να εισάγει μια **αλλαγή γραμμής (line break)** μέσα στο κείμενο.

Το `<br>` είναι ένα **κενό στοιχείο (empty element)**, δηλαδή **δεν περιέχει περιεχόμενο και δεν διαθέτει tag κλεισίματος**.

Χρησιμοποιείται όταν θέλετε το κείμενο να συνεχίσει στην επόμενη γραμμή, χωρίς να ξεκινήσει νέα παράγραφος.

## Παράδειγμα

```html
<!DOCTYPE html>
<html lang="el">
<head>
    <title>Διεύθυνση και Ποίημα</title>
</head>
<body>

    <h2>Διεύθυνση</h2>

    <p>
        Γιάννης Παπαδόπουλος<br>
        Οδός Κεντρική 123<br>
        Αθήνα, 10558<br>
        Ελλάδα
    </p>

    <h2>Ποίημα</h2>

    <p>
        Τα τριαντάφυλλα είναι κόκκινα,<br>
        Οι βιολέτες είναι μπλε,<br>
        Η ζάχαρη είναι γλυκιά,<br>
        Κι εσύ το ίδιο.
    </p>

</body>
</html>
```

> **Σημείωση:** Το `<br>` χρησιμοποιείται κυρίως σε περιπτώσεις όπως διευθύνσεις, ποιήματα, στίχους τραγουδιών ή οποιοδήποτε κείμενο όπου η αλλαγή γραμμής έχει σημασία χωρίς να δημιουργείται νέα παράγραφος.
