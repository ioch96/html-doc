# Σχόλιο HTML `<!-- ... -->`

Η σύνταξη σχολίων της HTML `<!-- ... -->` χρησιμοποιείται για την προσθήκη σχολίων στον πηγαίο κώδικα.

Τα σχόλια **δεν εμφανίζονται** στον φυλλομετρητή (browser). Είναι ορατά μόνο στον πηγαίο κώδικα της HTML και αγνοούνται από τον browser κατά την απόδοση (rendering) της σελίδας.

## Σύνταξη

```html
<!-- Αυτό είναι ένα σχόλιο -->
```

## Παράδειγμα 1

```html
<!DOCTYPE html>
<html lang="el">
  <head>
    <title>Τίτλος του Εγγράφου</title>
  </head>
  <body>
    <h1>Αυτή είναι μια επικεφαλίδα</h1>

    <!-- Αυτό είναι ένα σχόλιο. Τα σχόλια δεν εμφανίζονται στον browser. -->

    <p>Αυτή είναι μια παράγραφος.</p>
  </body>
</html>
```

## Πού είναι χρήσιμα τα σχόλια

### Επεξήγηση του κώδικα

Τα σχόλια χρησιμοποιούνται για να εξηγούν τι κάνει ένα τμήμα του κώδικα. Αυτό είναι ιδιαίτερα χρήσιμο όταν ο κώδικας είναι μεγάλος ή όταν συνεργάζεστε με άλλους προγραμματιστές.

#### Παράδειγμα 2

```html
<!-- Κύρια επικεφαλίδα της ιστοσελίδας -->
<h1>My Website</h1>

<!-- Παράγραφος καλωσορίσματος -->
<p>Καλώς ήρθατε στην ιστοσελίδα μας.</p>
```

---

### Προσθήκη σημειώσεων ή υπενθυμίσεων

Τα σχόλια μπορούν να χρησιμοποιηθούν ως προσωπικές σημειώσεις ή υπενθυμίσεις για μελλοντικές αλλαγές.

#### Παράδειγμα 3

```html
<!-- TODO: Προσθήκη φόρμας επικοινωνίας -->

<section>

  <h2>Επικοινωνία</h2>

</section>
```

Το `TODO` είναι ένας συνηθισμένος τρόπος για να δηλώσουμε ότι κάτι πρέπει να υλοποιηθεί αργότερα.

---

### Προσωρινή απενεργοποίηση κώδικα

Κατά τη διάρκεια δοκιμών (testing) ή αποσφαλμάτωσης (debugging), μπορούμε να "απενεργοποιήσουμε" προσωρινά κάποιο τμήμα του κώδικα χωρίς να το διαγράψουμε.

#### Παράδειγμα 4

```html
<h1>Η ιστοσελίδα μου</h1>

<!--
<p>
  Αυτή η παράγραφος δεν θα εμφανιστεί,
  επειδή βρίσκεται μέσα σε σχόλιο.
</p>
-->

<p>Αυτή η παράγραφος εμφανίζεται κανονικά.</p>
```

Ο browser αγνοεί οτιδήποτε βρίσκεται μέσα στα tags `<!--` και `-->`, επομένως το σχολιασμένο περιεχόμενο δεν εμφανίζεται στη σελίδα.
