# Δήλωση HTML `<!DOCTYPE>`

Κάθε έγγραφο HTML θα πρέπει να ξεκινά με μια δήλωση `<!DOCTYPE>`.

Η δήλωση `<!DOCTYPE>` **δεν** είναι HTML tag. Αντίθετα, ενημερώνει τον φυλλομετρητή (browser) για την έκδοση της HTML που χρησιμοποιεί το έγγραφο, ώστε να αποδώσει (render) σωστά τη σελίδα.

Στην **HTML5**, η δήλωση είναι πολύ απλή:

```html
<!DOCTYPE html>
```

> **Σημείωση:** Η δήλωση `<!DOCTYPE>` **δεν κάνει διάκριση μεταξύ πεζών και κεφαλαίων χαρακτήρων** (case-insensitive). Επομένως, τόσο το `<!DOCTYPE html>` όσο και το `<!doctype html>` είναι έγκυρα. Ωστόσο, το `<!DOCTYPE html>` αποτελεί την τυπική και πιο συχνά χρησιμοποιούμενη μορφή.

## Παράδειγμα

```html
<!DOCTYPE html>
<html lang="el">
  <head>
    <title>Τίτλος του Εγγράφου</title>
  </head>
  <body>
    <h1>Αυτή είναι μια επικεφαλίδα</h1>
    <p>Αυτή είναι μια παράγραφος.</p>
  </body>
</html>
```
