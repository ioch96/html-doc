# HTML `<div>` και `<span>`

Τα στοιχεία **`<div>`** και **`<span>`** χρησιμοποιούνται για την ομαδοποίηση (grouping) περιεχομένου σε μια ιστοσελίδα. Από μόνα τους **δεν έχουν κάποια συγκεκριμένη σημασιολογική σημασία (semantic meaning)** και χρησιμοποιούνται κυρίως για την εφαρμογή μορφοποίησης (CSS), τη διαχείριση της διάταξης (layout) και την αλληλεπίδραση μέσω JavaScript.

Η βασική διαφορά τους είναι ότι το **`<div>`** είναι **block-level element**, ενώ το **`<span>`** είναι **inline element**.

---

## Το στοιχείο `<div>`

Το `<div>` χρησιμοποιείται για την ομαδοποίηση μεγαλύτερων ενοτήτων περιεχομένου, όπως επικεφαλίδες, παραγράφους, εικόνες, φόρμες ή ακόμη και ολόκληρα τμήματα μιας ιστοσελίδας.

Το `<div>` δημιουργεί **νέα γραμμή** πριν και μετά από το περιεχόμενό του και καταλαμβάνει **ολόκληρο το διαθέσιμο πλάτος** του γονικού στοιχείου.

### Σύνταξη `<div>`

```html
<div>
    Περιεχόμενο
</div>
```

### Παράδειγμα `<div>`

```html
<div>
    <h2>Τίτλος</h2>
    <p>Αυτή είναι μια παράγραφος.</p>
</div>
```

---

## Το στοιχείο `<span>`

Το `<span>` χρησιμοποιείται για την ομαδοποίηση μικρών τμημάτων κειμένου ή άλλων inline στοιχείων.

Σε αντίθεση με το `<div>`, το `<span>` **δεν δημιουργεί νέα γραμμή** και καταλαμβάνει μόνο τον απαραίτητο χώρο που χρειάζεται το περιεχόμενό του.

### Σύνταξη `<span>`

```html
<span>Κείμενο</span>
```

### Παράδειγμα `<span>`

```html
<p>
    Η HTML είναι μια
    <span>γλώσσα σήμανσης</span>
    ιστοσελίδων.
</p>
```

Το `<span>` εμφανίζεται μέσα στην ίδια γραμμή της παραγράφου.

---

## Βασικές διαφορές

|Χαρακτηριστικό|`<div>`|`<span>`|
|:-:|:-:|:-:|
|Τύπος στοιχείου|Block|Inline|
|Ξεκινά νέα γραμμή|✔️|❌|
|Καταλαμβάνει όλο το διαθέσιμο πλάτος|✔️|❌|
|Χρήση|Ομαδοποίηση μεγάλων ενοτήτων|Ομαδοποίηση μικρών τμημάτων κειμένου|
|Μπορεί να περιέχει block στοιχεία|✔️|❌|
|Μπορεί να περιέχει inline στοιχεία|✔️|✔️|

---

## Παραδείγματα χρήσης

### Παράδειγμα 1 - Χρήση του `<div>`

```html
<div class="container">
    <h1>HTML</h1>
    <p>Η HTML χρησιμοποιείται για τη δημιουργία ιστοσελίδων.</p>
</div>
```

Το `<div>` χρησιμοποιείται ως ένα δοχείο (container) που περιέχει ολόκληρη την ενότητα.

---

### Παράδειγμα 2 - Χρήση του `<span>`

```html
<p>
    Η HTML είναι
    <span style="color:red;">εύκολη</span>
    στην εκμάθηση.
</p>
```

Το `<span>` χρησιμοποιείται μόνο για να μορφοποιήσει τη λέξη **εύκολη**.

---

### Παράδειγμα 3 - Πολλαπλά `<span>`

```html
<p>
    <span style="color:red;">Κόκκινο</span>,
    <span style="color:green;">Πράσινο</span>,
    <span style="color:blue;">Μπλε</span>
</p>
```

Κάθε `<span>` μορφοποιεί ανεξάρτητα ένα τμήμα του κειμένου.

---

### Παράδειγμα 4 - Διάταξη με `<div>`

```html
<div id="header">
    Header
</div>

<div id="main">
    Main Content
</div>

<div id="footer">
    Footer
</div>
```

Το `<div>` χρησιμοποιείται συχνά για τη δημιουργία της βασικής δομής μιας ιστοσελίδας.

---

## Συνδυασμός `<div>` και `<span>`

Τα δύο στοιχεία χρησιμοποιούνται συχνά μαζί.

```html
<div class="article">

    <h2>
        Μάθημα
        <span style="color:red;">HTML</span>
    </h2>

    <p>
        Η
        <span style="font-weight:bold;">HTML</span>
        αποτελεί τη βασική γλώσσα δημιουργίας ιστοσελίδων.
    </p>

</div>
```

Στο παραπάνω παράδειγμα:

- Το `<div>` ομαδοποιεί ολόκληρο το άρθρο.
- Τα `<span>` χρησιμοποιούνται για τη μορφοποίηση συγκεκριμένων λέξεων.

---

## Πότε χρησιμοποιούμε το καθένα

Χρησιμοποιούμε **`<div>`** όταν θέλουμε:

- να δημιουργήσουμε μια νέα ενότητα της σελίδας,
- να οργανώσουμε το layout,
- να εφαρμόσουμε CSS ή JavaScript σε μια ομάδα στοιχείων,
- να ομαδοποιήσουμε πολλά HTML στοιχεία.

Χρησιμοποιούμε **`<span>`** όταν θέλουμε:

- να μορφοποιήσουμε μία ή περισσότερες λέξεις,
- να εφαρμόσουμε CSS σε μικρό τμήμα κειμένου,
- να προσθέσουμε JavaScript σε ένα συγκεκριμένο κομμάτι μιας πρότασης,
- να μην αλλάξει η ροή του κειμένου.

---

## Best Practices

- Χρησιμοποιείτε το **`<div>`** για τη δημιουργία δοχείων (containers) και μεγάλων ενοτήτων της σελίδας.
- Χρησιμοποιείτε το **`<span>`** για τη μορφοποίηση ή τον χειρισμό μικρών τμημάτων κειμένου.
- Προτιμήστε τα **σημασιολογικά στοιχεία (semantic elements)**, όπως `<header>`, `<main>`, `<section>`, `<article>` και `<footer>`, όταν αυτά περιγράφουν καλύτερα το περιεχόμενο. Χρησιμοποιήστε το `<div>` μόνο όταν δεν υπάρχει κάποιο καταλληλότερο σημασιολογικό στοιχείο.
- Αποφύγετε την υπερβολική χρήση πολλών εμφωλευμένων (`nested`) `<div>`, καθώς δυσκολεύουν την ανάγνωση και τη συντήρηση του κώδικα.

---

## Συμπέρασμα

Τα στοιχεία **`<div>`** και **`<span>`** αποτελούν δύο από τα πιο συχνά χρησιμοποιούμενα στοιχεία της HTML. Παρότι δεν προσθέτουν κάποια σημασιολογική πληροφορία στο έγγραφο, είναι ιδιαίτερα χρήσιμα για την ομαδοποίηση περιεχομένου, την εφαρμογή κανόνων CSS και την αλληλεπίδραση μέσω JavaScript. Το **`<div>`** χρησιμοποιείται για μεγαλύτερες ενότητες και είναι στοιχείο τύπου **block**, ενώ το **`<span>`** χρησιμοποιείται για μικρά τμήματα περιεχομένου και είναι στοιχείο τύπου **inline**.
