# Εισαγωγή στην HTML

## Τι είναι η HTML

Η **HTML (HyperText Markup Language)** είναι η βασική γλώσσα σήμανσης που χρησιμοποιείται για τη δημιουργία και την οργάνωση του περιεχομένου μιας ιστοσελίδας.

Κάθε ιστοσελίδα που επισκεπτόμαστε στο διαδίκτυο περιέχει HTML. Η HTML περιγράφει τη δομή ενός εγγράφου και καθορίζει τον ρόλο που έχει κάθε τμήμα του περιεχομένου, ώστε ο browser να γνωρίζει πώς πρέπει να το ερμηνεύσει.

Για παράδειγμα, με την HTML μπορούμε να δηλώσουμε ότι ένα τμήμα του περιεχομένου αποτελεί:

* τίτλο,
* παράγραφο,
* εικόνα,
* υπερσύνδεσμο,
* πίνακα,
* λίστα,
* φόρμα,
* ή οποιοδήποτε άλλο δομικό στοιχείο μιας ιστοσελίδας.

Η HTML **δεν καθορίζει την εμφάνιση** των στοιχείων αυτών. Ο ρόλος της είναι να περιγράφει τη δομή και τη σημασία του περιεχομένου.

---

## Λίγα λόγια για την ιστορία της HTML

Η HTML δημιουργήθηκε το **1991** από τον **Tim Berners-Lee**, τον δημιουργό του **World Wide Web**.

Αρχικός σκοπός της ήταν η δημιουργία εγγράφων που μπορούσαν να συνδέονται μεταξύ τους μέσω υπερσυνδέσμων (hyperlinks), επιτρέποντας στους χρήστες να μεταβαίνουν εύκολα από ένα έγγραφο σε ένα άλλο.

Με την πάροδο των χρόνων η HTML εξελίχθηκε σημαντικά και σήμερα η **HTML5** αποτελεί το σύγχρονο πρότυπο για την ανάπτυξη ιστοσελίδων και web εφαρμογών.

---

## HTML, CSS και JavaScript

Οι περισσότερες σύγχρονες ιστοσελίδες βασίζονται στη συνεργασία τριών τεχνολογιών:

| Τεχνολογία     | Ρόλος                                            |
| -------------- | ------------------------------------------------ |
| **HTML**       | Περιγράφει τη δομή του περιεχομένου              |
| **CSS**        | Καθορίζει την εμφάνιση και τη μορφοποίηση        |
| **JavaScript** | Προσθέτει δυναμική συμπεριφορά και αλληλεπίδραση |

Μπορούμε να τις φανταστούμε σαν ένα σπίτι:

* Η **HTML** είναι ο σκελετός και οι τοίχοι.
* Η **CSS** είναι τα χρώματα, τα υλικά και η διακόσμηση.
* Η **JavaScript** είναι ο ηλεκτρισμός, οι αυτοματισμοί και όλα όσα κάνουν το σπίτι να "ζωντανεύει".

---

## Η HTML δεν είναι γλώσσα προγραμματισμού

Παρότι πολλοί την αποκαλούν λανθασμένα γλώσσα προγραμματισμού, η HTML **δεν είναι**.

Μια γλώσσα προγραμματισμού, όπως η JavaScript, η Python ή η Java, μπορεί να εκτελεί λογικές διαδικασίες, όπως:

* υπολογισμούς,
* επαναλήψεις,
* συνθήκες,
* μεταβλητές,
* συναρτήσεις,
* επεξεργασία δεδομένων,
* λήψη αποφάσεων.

Η HTML δεν διαθέτει καμία από αυτές τις δυνατότητες.

Ο μοναδικός της σκοπός είναι να περιγράφει τη δομή και τη σημασία του περιεχομένου.

Για τον λόγο αυτό χαρακτηρίζεται ως **γλώσσα σήμανσης (Markup Language)**.

---

## Τι σημαίνουν τα αρχικά HTML

Τα αρχικά **HTML** προέρχονται από τη φράση:

> **HyperText Markup Language**

Στα ελληνικά αποδίδεται συνήθως ως:

> **Γλώσσα Σήμανσης Υπερκειμένου**

Κάθε μία από τις τρεις λέξεις περιγράφει μια διαφορετική πτυχή της HTML.

---

### HyperText

Η λέξη **HyperText** σημαίνει **υπερκείμενο**.

Σε αντίθεση με ένα απλό κείμενο, το υπερκείμενο μπορεί να περιέχει υπερσυνδέσμους (hyperlinks), οι οποίοι συνδέουν διαφορετικά έγγραφα ή διαφορετικά σημεία του ίδιου εγγράφου.

Ένας υπερσύνδεσμος μπορεί να οδηγεί:

* σε άλλη ιστοσελίδα,
* σε διαφορετική σελίδα του ίδιου ιστοτόπου,
* σε συγκεκριμένο σημείο της ίδιας σελίδας,
* σε εικόνα,
* σε αρχείο,
* σε βίντεο,
* ή σε μια διεύθυνση email.

Παράδειγμα:

```html
<a href="https://example.com">Επισκέψου τη σελίδα</a>
```

---

### Markup

Η λέξη **Markup** σημαίνει **σήμανση**.

Η HTML χρησιμοποιεί ειδικές ετικέτες (**tags**) για να χαρακτηρίζει το περιεχόμενο.

Για παράδειγμα:

```html
<h1>Καλώς ήρθατε</h1>
```

δηλώνει ότι το κείμενο αποτελεί επικεφαλίδα.

```html
<p>Αυτή είναι μια παράγραφος.</p>
```

δηλώνει μια παράγραφο.

```html
<img src="photo.jpg" alt="Μια φωτογραφία">
```

δηλώνει μια εικόνα.

Ο browser δεν ενδιαφέρεται μόνο για το ποιο είναι το περιεχόμενο, αλλά και για το **τι αντιπροσωπεύει**.

---

### Language

Η λέξη **Language** σημαίνει **γλώσσα**.

Η HTML διαθέτει συγκεκριμένους κανόνες σύνταξης και γραμματικής, τους οποίους οι browsers μπορούν να κατανοήσουν και να επεξεργαστούν.

Ακολουθώντας αυτούς τους κανόνες, ο browser είναι σε θέση να δημιουργήσει τη δομή της ιστοσελίδας και να εμφανίσει το τελικό αποτέλεσμα στον χρήστη.

---

## Τι μπορούμε να δημιουργήσουμε με την HTML

Η HTML χρησιμοποιείται για τη δημιουργία του βασικού σκελετού μιας ιστοσελίδας.

Με αυτή μπορούμε να δημιουργήσουμε:

* επικεφαλίδες,
* παραγράφους,
* συνδέσμους,
* εικόνες,
* πίνακες,
* λίστες,
* φόρμες,
* κουμπιά,
* ενότητες,
* άρθρα,
* κεφαλίδες και υποσέλιδα,
* βίντεο,
* ηχητικό περιεχόμενο,
* καθώς και πολλά ακόμη δομικά στοιχεία.

---

## Τι δεν μπορεί να κάνει η HTML

Η HTML από μόνη της **δεν μπορεί**:

* να εκτελέσει υπολογισμούς,
* να δημιουργήσει μεταβλητές,
* να εκτελέσει επαναλήψεις,
* να αξιολογήσει συνθήκες,
* να αποθηκεύσει δεδομένα,
* να επικοινωνήσει με μια βάση δεδομένων,
* να δημιουργήσει διαδραστική συμπεριφορά.

Για αυτές τις δυνατότητες χρησιμοποιούνται άλλες τεχνολογίες, όπως η JavaScript και γλώσσες που εκτελούνται στον διακομιστή (server).

---

## Σύνοψη

Σε αυτό το κεφάλαιο μάθαμε ότι:

* Η HTML είναι η βασική γλώσσα σήμανσης του Παγκόσμιου Ιστού.
* Περιγράφει τη δομή και τη σημασία του περιεχομένου μιας ιστοσελίδας.
* Δεν είναι γλώσσα προγραμματισμού.
* Συνεργάζεται με τη CSS και τη JavaScript για τη δημιουργία σύγχρονων ιστοσελίδων.
* Χρησιμοποιεί tags για να χαρακτηρίζει τα διάφορα στοιχεία ενός εγγράφου.
* Τα αρχικά HTML σημαίνουν **HyperText Markup Language**.
