# Το Emmet στο Visual Studio Code

Το **Emmet** είναι ένα εργαλείο που επιταχύνει τη συγγραφή κώδικα **HTML** και **CSS** μέσω της χρήσης σύντομων συντομογραφιών (abbreviations). Στο **Visual Studio Code** το Emmet είναι **ενσωματωμένο (built-in)**, δηλαδή περιλαμβάνεται από προεπιλογή και **δεν απαιτεί την εγκατάσταση κάποιου επιπλέον extension**.

Ο προγραμματιστής μπορεί να πληκτρολογήσει μια σύντομη συντομογραφία και, πατώντας συνήθως το πλήκτρο **Tab**, το Emmet δημιουργεί αυτόματα τον αντίστοιχο κώδικα. Με αυτόν τον τρόπο μειώνεται ο χρόνος συγγραφής, αποφεύγονται επαναλαμβανόμενες ενέργειες και αυξάνεται η παραγωγικότητα κατά την ανάπτυξη ιστοσελίδων.

## Το Emmet ως ενσωματωμένη λειτουργία

Σε παλαιότερους επεξεργαστές κώδικα, όπως το **Sublime Text** ή το **Atom**, το Emmet εγκαθίστατο ως ξεχωριστό πρόσθετο (plugin). Αντίθετα, στο **Visual Studio Code** αποτελεί ενσωματωμένη λειτουργία και είναι διαθέσιμο αμέσως μετά την εγκατάσταση του προγράμματος.

Για να χρησιμοποιηθεί, αρκεί να δημιουργηθεί ένα αρχείο HTML ή CSS και να πληκτρολογηθεί μια έγκυρη συντομογραφία.

## Πλεονεκτήματα του Emmet

Η χρήση του Emmet προσφέρει αρκετά πλεονεκτήματα:

- Ταχύτερη συγγραφή HTML και CSS.
- Μείωση των συντακτικών λαθών.
- Αυτόματη δημιουργία επαναλαμβανόμενων στοιχείων.
- Ευκολότερη δημιουργία σύνθετων δομών HTML.
- Αύξηση της παραγωγικότητας του προγραμματιστή.

## Παραδείγματα χρήσης

### Δημιουργία βασικού εγγράφου HTML5

Γράφουμε:

```text
!
```

και πατάμε **Tab**.

Το Emmet δημιουργεί αυτόματα:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>

</body>
</html>
```

---

### Δημιουργία στοιχείου HTML

Γράφουμε:

```text
p
```

και πατάμε **Tab**.

Παράγεται:

```html
<p></p>
```

---

### Δημιουργία στοιχείου με κλάση (class)

Γράφουμε:

```text
div.container
```

Το αποτέλεσμα είναι:

```html
<div class="container"></div>
```

---

### Δημιουργία στοιχείου με αναγνωριστικό (id)

Γράφουμε:

```text
div#header
```

Το αποτέλεσμα είναι:

```html
<div id="header"></div>
```

---

### Δημιουργία πολλών στοιχείων

Γράφουμε:

```text
li*5
```

Το Emmet δημιουργεί:

```html
<li></li>
<li></li>
<li></li>
<li></li>
<li></li>
```

---

### Δημιουργία σύνθετης δομής

Γράφουμε:

```text
ul>li*3
```

Το αποτέλεσμα είναι:

```html
<ul>
    <li></li>
    <li></li>
    <li></li>
</ul>
```

Στο παράδειγμα:

- Το σύμβολο `>` δημιουργεί σχέση γονέα-παιδιού (parent-child).
- Το `*3` δημιουργεί τρία στοιχεία `<li>`.

---

### Προσθήκη κειμένου

Γράφουμε:

```text
h2{Καλώς ήρθατε}
```

Το αποτέλεσμα είναι:

```html
<h2>Καλώς ήρθατε</h2>
```

## Βασικά σύμβολα του Emmet

|Σύμβολο|Περιγραφή|Παράδειγμα|
|---------|-----------|------------|
|`!`|Δημιουργία βασικού εγγράφου HTML5|`!`|
|`>`|Δημιουργία στοιχείου-παιδιού|`ul>li`|
|`+`|Δημιουργία αδελφών στοιχείων|`h1+p`|
|`*`|Επανάληψη στοιχείων|`li*5`|
|`.`|Προσθήκη κλάσης (class)|`div.container`|
|`#`|Προσθήκη αναγνωριστικού (id)|`div#menu`|
|`{}`|Προσθήκη κειμένου|`p{Hello}`|

## Δημιουργία δοκιμαστικού κειμένου (Lorem Ipsum)

Το Emmet παρέχει τη δυνατότητα αυτόματης δημιουργίας **δοκιμαστικού κειμένου (Lorem Ipsum)**, το οποίο χρησιμοποιείται συχνά κατά την ανάπτυξη ιστοσελίδων. Το κείμενο αυτό επιτρέπει στον προγραμματιστή ή τον σχεδιαστή να ελέγξει τη διάταξη και τη μορφοποίηση μιας σελίδας, χωρίς να χρειάζεται να έχει διαθέσιμο το πραγματικό περιεχόμενο.

Για τη δημιουργία ενός στοιχείου παραγράφου που περιέχει δοκιμαστικό κείμενο, γράφουμε:

```text
p>lorem
```

ή ισοδύναμα:

```text
p>lorem10
```

και πατάμε **Tab**.

Το Emmet δημιουργεί αυτόματα:

```html
<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</p>
```

Στην παραπάνω περίπτωση, η λέξη `lorem` δημιουργεί ένα προεπιλεγμένο κείμενο **Lorem Ipsum**, ενώ το στοιχείο `<p>` δημιουργείται αυτόματα ως γονικό στοιχείο.

### Ορισμός αριθμού λέξεων

Μπορούμε επίσης να καθορίσουμε πόσες λέξεις θα περιέχει το κείμενο.

Παράδειγμα:

```text
p>lorem20
```

Το αποτέλεσμα είναι μια παράγραφος με περίπου **20 λέξεις**.

Αντίστοιχα:

```text
p>lorem50
```

δημιουργεί μια παράγραφο περίπου **50 λέξεων**.

### Παράδειγμα

Συντόμευση:

```text
p>lorem15
```

Παραγόμενος κώδικας:

```html
<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Nemo, laboriosam, harum.</p>
```

## Πότε χρησιμοποιείται

Το Lorem Ipsum χρησιμοποιείται όταν:

- δεν έχει γραφτεί ακόμη το πραγματικό περιεχόμενο μιας ιστοσελίδας,
- γίνεται σχεδίαση (layout) της σελίδας,
- ελέγχεται η μορφοποίηση (CSS),
- δοκιμάζεται η εμφάνιση παραγράφων, καρτών, άρθρων ή άλλων στοιχείων περιεχομένου.

> **Σημείωση:** Το **Lorem Ipsum** αποτελεί κείμενο χωρίς πραγματικό νόημα. Χρησιμοποιείται αποκλειστικά ως προσωρινό περιεχόμενο (placeholder text) κατά την ανάπτυξη και τον σχεδιασμό ιστοσελίδων.
