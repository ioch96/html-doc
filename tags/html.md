# HTML `<html>` Tag

Το tag `<html>` αντιπροσωπεύει το **ριζικό στοιχείο (root element)** ενός εγγράφου HTML.

Όλα τα υπόλοιπα στοιχεία της HTML (εκτός από τη δήλωση `<!DOCTYPE>`) βρίσκονται μέσα στο στοιχείο `<html>`.

Με άλλα λόγια, το `<html>` αποτελεί το **κύριο container** που περιέχει ολόκληρη τη δομή της ιστοσελίδας.

> **Σημείωση:** Συνιστάται να χρησιμοποιείτε πάντα το attribute `lang` στο tag `<html>` για να δηλώνετε τη γλώσσα της ιστοσελίδας. Αυτό βοηθά τους browsers, τις μηχανές αναζήτησης (SEO) και τις τεχνολογίες υποβοήθησης (όπως οι screen readers) να κατανοήσουν σωστά το περιεχόμενο.

## Σύνταξη

```html
<html lang="el">
  ...
</html>
```

## Παράδειγμα

```html
<!DOCTYPE html>
<html lang="el">
  <head>
    <title>Τίτλος του Εγγράφου</title>
  </head>

  <body>
    <h1>Αυτή είναι μια επικεφαλίδα</h1>
    <p>Αυτή είναι μια παράγραφος.</p>
  </body>
</html>
```

## Το attribute `lang`

Το attribute `lang` καθορίζει τη γλώσσα του περιεχομένου της ιστοσελίδας.

Παραδείγματα:

```html
<html lang="el">
```

Ελληνικά.

```html
<html lang="en">
```

Αγγλικά.

```html
<html lang="fr">
```

Γαλλικά.

```html
<html lang="de">
```

Γερμανικά.

## Δομή ενός εγγράφου HTML

Το στοιχείο `<html>` περιέχει συνήθως δύο βασικά τμήματα:

```text
<html>
├── <head>
│   ├── <title>
│   ├── <meta>
│   ├── <link>
│   └── ...
└── <body>
    ├── <h1>
    ├── <p>
    ├── <img>
    ├── <a>
    └── ...
</html>
```

- Το `<head>` περιέχει πληροφορίες σχετικά με το έγγραφο (metadata), οι οποίες **δεν εμφανίζονται** στη σελίδα.
- Το `<body>` περιέχει όλο το **ορατό περιεχόμενο** της ιστοσελίδας.

## Βέλτιστες Πρακτικές (Best Practices)

- Ξεκινάτε πάντα το έγγραφο με τη δήλωση `<!DOCTYPE html>`.
- Χρησιμοποιείτε πάντα το attribute `lang`.
- Τοποθετείτε μέσα στο `<html>` μόνο τα στοιχεία `<head>` και `<body>`.
- Διατηρείτε σωστή εσοχή (indentation) για καλύτερη αναγνωσιμότητα του κώδικα.

> **Σημείωση:** Το στοιχείο `<html>` είναι το **ανώτερο επίπεδο** της δομής ενός εγγράφου HTML. Όλα τα υπόλοιπα στοιχεία της ιστοσελίδας βρίσκονται μέσα σε αυτό.
