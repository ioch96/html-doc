# HTML `<mark>` Tag

Το tag `<mark>` χρησιμοποιείται για να ορίσει κείμενο που πρέπει να **επισημανθεί (marked)** ή να **τονιστεί με επισήμανση (highlighted)**.

Στους περισσότερους browsers, το περιεχόμενό του εμφανίζεται με **κίτρινο φόντο**, όπως όταν επισημαίνουμε κείμενο με έναν μαρκαδόρο.

Το `<mark>` χρησιμοποιείται όταν θέλουμε να δείξουμε ότι ένα συγκεκριμένο τμήμα του κειμένου έχει ιδιαίτερη σημασία στο συγκεκριμένο πλαίσιο, χωρίς όμως να δηλώνουμε ότι είναι πιο σημαντικό από το υπόλοιπο κείμενο.

## Παράδειγμα

```html
<!DOCTYPE html>
<html lang="el">
  <head>
    <title>Τίτλος του Εγγράφου</title>
  </head>

  <body>

    <h1>Το στοιχείο mark</h1>

    <p>
      Μην ξεχάσετε να ολοκληρώσετε την
      <mark>εργασία σας μέχρι αύριο</mark>.
    </p>

  </body>
</html>
```

## Συνήθεις χρήσεις

Το `<mark>` χρησιμοποιείται συχνά για:

### Επισήμανση αποτελεσμάτων αναζήτησης

```html
<p>
  Βρέθηκαν αποτελέσματα για τη λέξη
  <mark>HTML</mark>.
</p>
```

### Ανάδειξη σημαντικού σημείου

```html
<p>
  Η προθεσμία λήγει στις
  <mark>30 Ιουνίου</mark>.
</p>
```

### Επισήμανση μέσα σε άρθρα ή σημειώσεις

```html
<p>
  Το στοιχείο <mark>section</mark> χρησιμοποιείται
  για τον ορισμό μιας θεματικής ενότητας.
</p>
```

## Βέλτιστες Πρακτικές (Best Practices)

- Χρησιμοποιείτε το `<mark>` όταν θέλετε να επισημάνετε πληροφορίες που είναι σημαντικές.
- Είναι ιδιαίτερα χρήσιμο για αποτελέσματα αναζήτησης, υπενθυμίσεις και επισημάνσεις μέσα σε κείμενα.
- Αν θέλετε να δώσετε σημασιολογική έμφαση ή σπουδαιότητα, χρησιμοποιήστε τα `<em>` ή `<strong>` αντί για το `<mark>`.

> **Σημείωση:** Το `<mark>` είναι ένα **σημασιολογικό στοιχείο (semantic element)**. Δεν χρησιμοποιείται μόνο για να προσθέσει χρώμα στο κείμενο, αλλά για να δηλώσει ότι ένα συγκεκριμένο τμήμα του περιεχομένου πρέπει να ξεχωρίζει ή να τραβά την προσοχή του αναγνώστη.
