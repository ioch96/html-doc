# HTML `<li>` Tag

Το tag `<li>` χρησιμοποιείται για να ορίσει ένα **στοιχείο λίστας (list item)**.

Το `<li>` χρησιμοποιείται μέσα στις παρακάτω λίστες:

- `<ul>` (μη ταξινομημένη λίστα)
- `<ol>` (ταξινομημένη λίστα)
- `<menu>` (λίστα μενού)

Σε μια λίστα `<ul>` ή `<menu>`, τα στοιχεία εμφανίζονται συνήθως με **κουκκίδες (bullets)**.

Σε μια λίστα `<ol>`, τα στοιχεία εμφανίζονται συνήθως με **αριθμούς**, **γράμματα** ή **ρωμαϊκούς αριθμούς**.

## Σύνταξη

### Μέσα σε μη ταξινομημένη λίστα

```html
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
```

### Μέσα σε ταξινομημένη λίστα

```html
<ol>
  <li>Εγκατάσταση Node.js</li>
  <li>Εγκατάσταση React</li>
  <li>Δημιουργία Project</li>
</ol>
```

## Παράδειγμα 1

```html
<!DOCTYPE html>
<html lang="el">
<head>
  <title>Τίτλος του Εγγράφου</title>
</head>

<body>

  <h1>Το στοιχείο li</h1>

  <h2>Μη ταξινομημένη λίστα</h2>

  <ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
  </ul>

  <h2>Ταξινομημένη λίστα</h2>

  <ol>
    <li>Πρώτο βήμα</li>
    <li>Δεύτερο βήμα</li>
    <li>Τρίτο βήμα</li>
  </ol>

</body>
</html>
```

## Attribute

|Attribute|Τιμή|Περιγραφή|
|:---------:|:----:|-----------|
|`value`|`number`|Χρησιμοποιείται μόνο μέσα σε λίστες `<ol>`. Καθορίζει τον αριθμό από τον οποίο θα ξεκινήσει το συγκεκριμένο στοιχείο της λίστας. Τα επόμενα στοιχεία συνεχίζουν την αρίθμηση από εκεί.|

## Το attribute `value`

Το `value` επιτρέπει να αλλάξετε την αρίθμηση ενός συγκεκριμένου στοιχείου μέσα σε μια αριθμημένη λίστα.

### Παράδειγμα 2

```html
<ol>
  <li>HTML</li>
  <li>CSS</li>
  <li value="10">JavaScript</li>
  <li>React</li>
</ol>
```

### Αποτέλεσμα 2

```text
1. HTML
2. CSS
10. JavaScript
11. React
```

## Εμφωλευμένες λίστες (Nested Lists)

Ένα στοιχείο `<li>` μπορεί να περιέχει και άλλη λίστα.

```html
<ul>
  <li>
    Frontend

    <ul>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>
  </li>

  <li>Backend</li>
</ul>
```

## Τι μπορεί να περιέχει ένα `<li>`

Ένα στοιχείο `<li>` δεν περιορίζεται μόνο σε απλό κείμενο. Μπορεί να περιέχει σχεδόν οποιοδήποτε HTML στοιχείο.

Παράδειγμα:

```html
<ul>
  <li>
    <strong>HTML</strong>

    <p>Η βασική γλώσσα του Web.</p>

    <a href="#">Περισσότερα...</a>
  </li>
</ul>
```

## Βέλτιστες Πρακτικές (Best Practices)

- Χρησιμοποιείτε το `<li>` μόνο μέσα σε στοιχεία `<ul>`, `<ol>` ή `<menu>`.
- Κάθε στοιχείο της λίστας πρέπει να βρίσκεται μέσα σε ξεχωριστό `<li>`.
- Για υπολίστες, τοποθετήστε τη νέα λίστα μέσα στο αντίστοιχο `<li>`.

> **Σημείωση:** Το `<li>` είναι ένα **σημασιολογικό στοιχείο (semantic element)** που δηλώνει ότι το περιεχόμενό του αποτελεί ένα στοιχείο μιας λίστας. Σε συνδυασμό με τα `<ul>` και `<ol>` χρησιμοποιείται πολύ συχνά σε μενού πλοήγησης, λίστες προϊόντων και λίστες εργασιών (to-do lists).
