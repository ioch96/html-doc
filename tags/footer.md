# HTML `<footer>` Tag

Το tag `<footer>` χρησιμοποιείται για να ορίσει το **υποσέλιδο (footer)** μιας ιστοσελίδας ή μιας θεματικής ενότητας.

Το `<footer>` είναι ένα **σημασιολογικό στοιχείο (semantic element)** που συνήθως περιέχει πληροφορίες σχετικές με το περιεχόμενο στο οποίο ανήκει.

## Παράδειγμα

```html
<!DOCTYPE html>
<html lang="el">
<head>
  <title>Το στοιχείο footer</title>
</head>

<body>

  <main>

    <h1>HTML Footer</h1>

    <p>
      Αυτό είναι το κύριο περιεχόμενο της σελίδας.
    </p>

  </main>

  <footer>

    <p>&copy; 2026 My Website. All rights reserved.</p>

  </footer>

</body>
</html>
```

## Το `<footer>` μέσα σε ένα `<article>`

Το `<footer>` δεν χρησιμοποιείται μόνο στο τέλος μιας ιστοσελίδας.

Μπορεί επίσης να χρησιμοποιηθεί στο τέλος ενός άρθρου (`<article>`) ή μιας ενότητας (`<section>`).

Παράδειγμα:

```html
<article>

  <h2>Τι είναι η HTML;</h2>

  <p>
    Η HTML είναι η βασική γλώσσα σήμανσης του Web.
  </p>

  <footer>

    <p>Συγγραφέας: Γιάννης Παπαδόπουλος</p>

    <p>Δημοσιεύτηκε στις 16 Ιουλίου 2026</p>

  </footer>

</article>
```

Εδώ το `<footer>` περιέχει πληροφορίες που αφορούν **μόνο το συγκεκριμένο άρθρο**.

---

## Συνήθεις χρήσεις

Το `<footer>` χρησιμοποιείται συχνά για:

- Copyright
- Πληροφορίες συγγραφέα
- Στοιχεία επικοινωνίας
- Πολιτική Απορρήτου (Privacy Policy)
- Όρους Χρήσης (Terms of Use)
- Sitemap
- Social Media Links
- Σύνδεσμο «Επιστροφή στην κορυφή»

Παράδειγμα:

```html
<footer>

  <p>&copy; 2026 My Website</p>

  <a href="#">Πολιτική Απορρήτου</a>

  <a href="#">Όροι Χρήσης</a>

  <a href="#top">Επιστροφή στην κορυφή</a>

</footer>
```

---

## Χρήση του `<address>` μέσα στο `<footer>`

Όταν το υποσέλιδο περιέχει στοιχεία επικοινωνίας, είναι καλή πρακτική να χρησιμοποιείται το στοιχείο `<address>`.

Παράδειγμα:

```html
<footer>

  <address>

    Email:
    <a href="mailto:info@example.com">
      info@example.com
    </a>

  </address>

</footer>
```

---

## Βέλτιστες Πρακτικές (Best Practices)

- Χρησιμοποιείτε το `<footer>` για πληροφορίες που αφορούν τη σελίδα ή την ενότητα στην οποία ανήκει.
- Μπορείτε να χρησιμοποιείτε περισσότερα από ένα `<footer>` στην ίδια σελίδα.
- Μην χρησιμοποιείτε το `<footer>` μόνο για λόγους μορφοποίησης. Αν χρειάζεστε απλώς ένα γενικό container, χρησιμοποιήστε το `<div>`.

> **Σημείωση:** Το `<footer>` είναι ένα **σημασιολογικό στοιχείο (semantic element)** που χρησιμοποιείται για να δηλώσει το υποσέλιδο μιας ιστοσελίδας ή μιας ενότητας. Συμβάλλει στην καλύτερη οργάνωση του κώδικα, βελτιώνει την προσβασιμότητα και βοηθά τις μηχανές αναζήτησης να κατανοήσουν τη δομή του εγγράφου.
