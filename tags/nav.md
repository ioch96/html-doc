# HTML `<nav>` Tag

Το tag `<nav>` χρησιμοποιείται για να ορίσει μια **ενότητα πλοήγησης (navigation section)** μιας ιστοσελίδας.

Περιέχει συνήθως τους **κύριους συνδέσμους (navigation links)** που βοηθούν τον χρήστη να μετακινείται μεταξύ των βασικών σελίδων ή ενοτήτων του ιστότοπου.

Το `<nav>` είναι ένα **σημασιολογικό στοιχείο (semantic element)**, καθώς δηλώνει ότι το περιεχόμενό του αποτελεί τμήμα πλοήγησης.

> **Σημείωση:** **Δεν πρέπει όλοι οι σύνδεσμοι (`<a>`) μιας ιστοσελίδας να βρίσκονται μέσα σε ένα `<nav>`.** Το `<nav>` προορίζεται μόνο για τις **κύριες ομάδες συνδέσμων πλοήγησης**, όπως το βασικό μενού ή η δευτερεύουσα πλοήγηση.

## Παράδειγμα

```html
<!DOCTYPE html>
<html lang="el">
<head>
  <title>Το στοιχείο nav</title>
</head>

<body>

  <header>

    <h1>My Website</h1>

    <nav>
      <a href="#">Αρχική</a>
      <a href="#">Άρθρα</a>
      <a href="#">Υπηρεσίες</a>
      <a href="#">Επικοινωνία</a>
    </nav>

  </header>

</body>
</html>
```

## Πότε χρησιμοποιούμε το `<nav>`

Το `<nav>` χρησιμοποιείται για:

- Το κύριο μενού μιας ιστοσελίδας.
- Το μενού στο υποσέλιδο (footer navigation).
- Το breadcrumb navigation.
- Την πλοήγηση μεταξύ σελίδων (pagination).
- Ένα sidebar με κύριους συνδέσμους.

Παράδειγμα:

```html
<nav>
  <ul>
    <li><a href="/">Αρχική</a></li>
    <li><a href="/about">Σχετικά</a></li>
    <li><a href="/blog">Blog</a></li>
    <li><a href="/contact">Επικοινωνία</a></li>
  </ul>
</nav>
```

## Πότε **δεν** χρησιμοποιούμε το `<nav>`

Δεν το χρησιμοποιούμε για οποιοδήποτε σύνολο συνδέσμων.

Για παράδειγμα:

```html
<p>
  Διαβάστε περισσότερα στο
  <a href="article.html">επόμενο άρθρο</a>.
</p>
```

ή

```html
<p>
  Ακολουθήστε μας στο
  <a href="#">Facebook</a>.
</p>
```

Αυτοί οι σύνδεσμοι **δεν αποτελούν κύρια πλοήγηση**, επομένως **δεν πρέπει** να τοποθετούνται μέσα σε `<nav>`.

## Χρήση με λίστες

Συνήθως οι σύνδεσμοι πλοήγησης οργανώνονται σε λίστα.

```html
<nav>
  <ul>
    <li><a href="#">Αρχική</a></li>
    <li><a href="#">Προϊόντα</a></li>
    <li><a href="#">Blog</a></li>
    <li><a href="#">Επικοινωνία</a></li>
  </ul>
</nav>
```

Αυτή είναι η πιο συνηθισμένη και προτεινόμενη πρακτική.

## Πλεονεκτήματα του `<nav>`

Η χρήση του `<nav>` προσφέρει:

- Καλύτερη οργάνωση του κώδικα.
- Βελτιωμένη προσβασιμότητα (Accessibility).
- Καλύτερη κατανόηση της δομής της σελίδας από τις μηχανές αναζήτησης (SEO).
- Ευκολότερη πλοήγηση για χρήστες που χρησιμοποιούν screen readers.

Οι screen readers μπορούν να αναγνωρίσουν το `<nav>` ως περιοχή πλοήγησης και να επιτρέψουν στον χρήστη να μεταβεί απευθείας σε αυτήν ή να την παραλείψει.

## Βέλτιστες Πρακτικές (Best Practices)

- Χρησιμοποιείτε το `<nav>` μόνο για τις κύριες περιοχές πλοήγησης.
- Οργανώνετε τους συνδέσμους με στοιχεία `<ul>` και `<li>`, όταν πρόκειται για μενού.
- Μπορείτε να έχετε περισσότερα από ένα στοιχεία `<nav>` στην ίδια σελίδα, εφόσον εξυπηρετούν διαφορετικούς σκοπούς (π.χ. κύριο μενού και breadcrumb navigation).
- Μην το χρησιμοποιείτε για μεμονωμένους συνδέσμους μέσα σε άρθρα ή παραγράφους.

> **Σημείωση:** Το `<nav>` είναι ένα από τα σημαντικότερα **σημασιολογικά στοιχεία (semantic elements)** της HTML. Χρησιμοποιείται σχεδόν σε κάθε σύγχρονη ιστοσελίδα και εφαρμογή για τον ορισμό των κύριων περιοχών πλοήγησης, βελτιώνοντας την αναγνωσιμότητα του κώδικα, την προσβασιμότητα και το SEO.
