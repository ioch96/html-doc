# HTML `<center>` Tag

> **Δεν υποστηρίζεται στην HTML5.**

Το tag `<center>` χρησιμοποιούνταν στην **HTML4** για τη **στοίχιση του περιεχομένου στο κέντρο** της σελίδας.

Στην **HTML5** έχει καταργηθεί (**deprecated**) και **δεν πρέπει πλέον να χρησιμοποιείται**.

## Παράδειγμα (HTML4)

```html
<!DOCTYPE html>
<html lang="el">
  <head>
    <title>Τίτλος του Εγγράφου</title>
  </head>
  <body>
    <center>
      <h1>Αυτή είναι μια επικεφαλίδα</h1>
      <p>Αυτή είναι μια παράγραφος.</p>
    </center>
  </body>
</html>
```

## Τι να χρησιμοποιήσετε αντί για αυτό;

Στην HTML5, η στοίχιση του κειμένου ή άλλων στοιχείων γίνεται με **CSS** και όχι με το tag `<center>`.

### Στοίχιση κειμένου στο κέντρο με CSS

```html
<!DOCTYPE html>
<html lang="el">
  <head>
    <title>Τίτλος του Εγγράφου</title>

    <style>
      h1 {
        text-align: center;
      }

      p {
        text-align: center;
      }
    </style>
  </head>

  <body>
    <h1>Αυτή είναι μια επικεφαλίδα</h1>
    <p>Αυτή είναι μια παράγραφος.</p>
  </body>
</html>
```

> **Σημείωση:** Η HTML χρησιμοποιείται για τη **δομή** του περιεχομένου, ενώ η CSS είναι υπεύθυνη για την **εμφάνιση και τη μορφοποίησή** του. Για αυτόν τον λόγο, η στοίχιση στο κέντρο πρέπει να γίνεται με την ιδιότητα CSS `text-align: center;` και όχι με το παρωχημένο tag `<center>`.
