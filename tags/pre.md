# HTML `<pre>` Tag

Το tag `<pre>` χρησιμοποιείται για να ορίσει **προμορφοποιημένο κείμενο (preformatted text)**.

Το κείμενο μέσα σε ένα στοιχείο `<pre>`:

- Εμφανίζεται με **γραμματοσειρά σταθερού πλάτους (fixed-width ή monospace)**.
- **Διατηρεί όλα τα κενά (spaces)** ακριβώς όπως έχουν γραφτεί.
- **Διατηρεί όλες τις αλλαγές γραμμής (line breaks)**.
- Εμφανίζεται ακριβώς όπως είναι γραμμένο στον πηγαίο κώδικα της HTML.

Αυτό το καθιστά ιδανικό για την εμφάνιση:

- Κώδικα προγραμμάτων
- Αρχείων ρυθμίσεων (configuration files)
- Ποιημάτων
- ASCII Art
- Οποιουδήποτε κειμένου όπου η μορφοποίηση πρέπει να διατηρηθεί.

## Παράδειγμα

```html
<!DOCTYPE html>
<html lang="el">
  <head>
    <title>Τίτλος του Εγγράφου</title>
  </head>

  <body>

    <h1>Το στοιχείο pre</h1>

    <pre>
Το κείμενο μέσα στο στοιχείο pre
εμφανίζεται με γραμματοσειρά
σταθερού πλάτους και διατηρεί

      όλα τα κενά
και τις αλλαγές γραμμής.
    </pre>

  </body>
</html>
```

## Πότε να χρησιμοποιείτε το `<pre>`

Το `<pre>` είναι ιδιαίτερα χρήσιμο όταν θέλετε να διατηρήσετε την ακριβή μορφοποίηση του κειμένου, όπως στις παρακάτω περιπτώσεις:

```html
<pre>
function greet(name) {
  console.log("Hello " + name);
}
</pre>
```

ή

```html
<pre>
      *
     ***
    *****
   *******
***********
</pre>
```

> **Σημείωση:** Το στοιχείο `<pre>` διατηρεί αυτόματα τα κενά και τις αλλαγές γραμμής, επομένως **δεν χρειάζεται** να χρησιμοποιήσετε το tag `<br>` ή HTML entities όπως `&nbsp;` για τη μορφοποίηση του περιεχομένου.
