# HTML `<main>` Tag

Το tag `<main>` χρησιμοποιείται για να ορίσει το **κύριο περιεχόμενο (main content)** μιας ιστοσελίδας.

Το περιεχόμενο μέσα στο `<main>` πρέπει να είναι **μοναδικό** για τη συγκεκριμένη σελίδα και να περιέχει τις πληροφορίες που αποτελούν το βασικό θέμα της.

Το `<main>` είναι ένα **σημασιολογικό στοιχείο (semantic element)** που βοηθά τους browsers, τις μηχανές αναζήτησης και τις τεχνολογίες υποβοήθησης να αναγνωρίζουν εύκολα ποιο είναι το κύριο περιεχόμενο της σελίδας.

## Τι **δεν** πρέπει να περιέχει

Το `<main>` **δεν πρέπει** να περιλαμβάνει περιεχόμενο που επαναλαμβάνεται σε πολλές σελίδες του ίδιου ιστότοπου, όπως:

- Μενού πλοήγησης (`<nav>`).
- Sidebar (`<aside>`).
- Copyright.
- Λογότυπο (logo).
- Φόρμες αναζήτησης.
- Υποσέλιδο (`<footer>`).

Αυτά τα στοιχεία ανήκουν συνήθως εκτός του `<main>`.

---

### Παράδειγμα

```html
<!DOCTYPE html>
<html lang="el">

<head>
  <title>Το στοιχείο main</title>
</head>

<body>

  <header>

    <h1>My Website</h1>

    <nav>
      <a href="#">Αρχική</a>
      <a href="#">Blog</a>
      <a href="#">Επικοινωνία</a>
    </nav>

  </header>

  <main>

    <h2>Τι είναι η HTML;</h2>

    <p>
      Η HTML είναι η βασική γλώσσα σήμανσης
      που χρησιμοποιείται για τη δημιουργία ιστοσελίδων.
    </p>

  </main>

  <footer>

    <p>&copy; 2026 My Website</p>

  </footer>

</body>

</html>
```

---

## Παράδειγμα με `<article>`

Το `<main>` συχνά περιέχει ένα ή περισσότερα στοιχεία `<article>`.

```html
<main>

  <article>

    <header>

      <h2>Τι είναι η HTML;</h2>

    </header>

    <p>
      Η HTML είναι η βασική γλώσσα σήμανσης του Web.
    </p>

  </article>

</main>
```

Εδώ:

- Το `<main>` περιέχει το κύριο περιεχόμενο της σελίδας.
- Το `<article>` περιέχει ένα αυτόνομο άρθρο.
- Το `<header>` περιέχει την κεφαλίδα του άρθρου.

---

## Πόσα `<main>` επιτρέπονται

Σε κάθε HTML έγγραφο επιτρέπεται **μόνο ένα** στοιχείο `<main>`.

```html
<body>

  <header>...</header>

  <main>
    ...
  </main>

  <footer>...</footer>

</body>
```

---

## Πού **δεν** επιτρέπεται να βρίσκεται

Το `<main>` **δεν πρέπει** να βρίσκεται μέσα σε:

- `<article>`
- `<aside>`
- `<footer>`
- `<header>`
- `<nav>`

---

## Γιατί χρησιμοποιούμε το `<main>`

Χωρίς Semantic HTML:

```html
<div id="content">

  <h1>HTML</h1>

  <p>...</p>

</div>
```

Ο browser δεν γνωρίζει ότι αυτό είναι το κύριο περιεχόμενο.

Με Semantic HTML:

```html
<main>

  <h1>HTML</h1>

  <p>...</p>

</main>
```

Τώρα είναι ξεκάθαρο ότι αυτό είναι το σημαντικότερο μέρος της σελίδας.

---

## Βέλτιστες Πρακτικές (Best Practices)

- Χρησιμοποιείτε **μόνο ένα** `<main>` ανά HTML έγγραφο.
- Τοποθετήστε μέσα του μόνο το βασικό περιεχόμενο της σελίδας.
- Μην τοποθετείτε μέσα στο `<main>` στοιχεία όπως το κύριο μενού, το λογότυπο ή το υποσέλιδο, εφόσον αυτά επαναλαμβάνονται σε όλες τις σελίδες.
- Χρησιμοποιείτε στοιχεία όπως `<article>` και `<section>` για να οργανώνετε το περιεχόμενο μέσα στο `<main>`.

> **Σημείωση:** Το `<main>` είναι ένα από τα σημαντικότερα **σημασιολογικά στοιχεία (semantic elements)** της HTML. Δηλώνει ποιο είναι το κύριο περιεχόμενο της ιστοσελίδας, βελτιώνοντας την προσβασιμότητα, το SEO και την αναγνωσιμότητα του κώδικα. Στις περισσότερες σύγχρονες ιστοσελίδες και εφαρμογές υπάρχει ένα μόνο `<main>` που περιέχει το βασικό περιεχόμενο της σελίδας.
