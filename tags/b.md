# HTML `<b>` Tag

Το tag `<b>` χρησιμοποιείται για να εμφανίσει το περιεχόμενό του με **έντονη γραφή (bold)**, **χωρίς να προσδίδει κάποια επιπλέον σημασιολογική έμφαση** στο κείμενο.

Με άλλα λόγια, το `<b>` αλλάζει μόνο την εμφάνιση του κειμένου και όχι τη σημασία του.

> **Σημείωση:** Αν θέλετε να δώσετε έμφαση ή σημασία στο κείμενο, προτιμήστε το tag `<strong>`. Το `<b>` χρησιμοποιείται κυρίως για οπτική επισήμανση.

## Παράδειγμα

```html
<p>Αυτό είναι κανονικό κείμενο - <b>και αυτό είναι έντονο κείμενο</b>.</p>
```
