# HTML `<head>` Tag

Το tag `<head>` είναι ένα **metadata container** και βρίσκεται ανάμεσα στα στοιχεία `<html>` και `<body>`.

Τα **metadata** είναι πληροφορίες που περιγράφουν το ίδιο το έγγραφο HTML. Οι πληροφορίες αυτές **δεν εμφανίζονται** στην ιστοσελίδα, αλλά χρησιμοποιούνται από τους browsers, τις μηχανές αναζήτησης και άλλες εφαρμογές.

Συνήθως, μέσα στο `<head>` ορίζονται:

- Ο τίτλος της σελίδας.
- Η κωδικοποίηση χαρακτήρων.
- Οι πληροφορίες για την προβολή σε κινητές συσκευές.
- Τα CSS αρχεία ή οι εσωτερικοί κανόνες CSS.
- Τα αρχεία JavaScript.
- Τα metadata για SEO και κοινωνικά δίκτυα.
- Άλλες πληροφορίες που αφορούν το έγγραφο.

## Στοιχεία που μπορούν να περιέχονται στο `<head>`

- `<title>` *(υποχρεωτικό σε κάθε έγγραφο HTML)*
- `<style>`
- `<base>`
- `<link>`
- `<meta>`
- `<script>`
- `<noscript>`

## Παράδειγμα

```html
<!DOCTYPE html>
<html lang="el">
<head>

  <!-- Υποχρεωτικό: Ο τίτλος του εγγράφου -->
  <title>Παράδειγμα HTML Head</title>

  <!-- Meta tags -->
  <meta charset="UTF-8">
  <meta
    name="viewport"
    content="width=device-width, initial-scale=1.0"
  >
  <meta
    name="description"
    content="Παράδειγμα όλων των στοιχείων του HTML head"
  >

  <!-- Βασική διεύθυνση για σχετικά URLs -->
  <base href="https://www.example.com/">

  <!-- Εξωτερικό αρχείο CSS -->
  <link rel="stylesheet" href="styles.css">

  <!-- Εσωτερικό CSS -->
  <style>
    body {
      font-family: Arial, sans-serif;
      background-color: #f0f0f0;
    }

    h1 {
      color: #333;
    }
  </style>

  <!-- Εξωτερικό αρχείο JavaScript -->
  <script src="script.js" defer></script>

  <!-- Εσωτερικός κώδικας JavaScript -->
  <script>
    console.log("Hello from inline script!");
  </script>

  <!-- Περιεχόμενο που εμφανίζεται όταν η JavaScript είναι απενεργοποιημένη -->
  <noscript>
    <p>
      Η JavaScript είναι απενεργοποιημένη στον browser σας.
      Ορισμένες λειτουργίες ενδέχεται να μη λειτουργούν σωστά.
    </p>
  </noscript>

</head>

<body>

  <h1>Παράδειγμα του στοιχείου Head</h1>

  <p>
    Αυτή η σελίδα παρουσιάζει τα βασικά στοιχεία που μπορούν
    να τοποθετηθούν μέσα στο στοιχείο head.
  </p>

</body>
</html>
```

## Τα σημαντικότερα στοιχεία του `<head>`

|Στοιχείο|Περιγραφή|
|:--------:|-----------|
|`<title>`|Ορίζει τον τίτλο της σελίδας που εμφανίζεται στην καρτέλα του browser και στα αποτελέσματα αναζήτησης.|
|`<meta>`|Περιέχει μεταδεδομένα, όπως η κωδικοποίηση χαρακτήρων, το viewport και πληροφορίες SEO.|
|`<link>`|Συνδέει το έγγραφο με εξωτερικούς πόρους, όπως αρχεία CSS ή favicon.|
|`<style>`|Περιέχει εσωτερικούς κανόνες CSS.|
|`<script>`|Περιέχει ή συνδέει κώδικα JavaScript.|
|`<noscript>`|Εμφανίζει εναλλακτικό περιεχόμενο όταν η JavaScript είναι απενεργοποιημένη.|
|`<base>`|Καθορίζει τη βασική διεύθυνση (base URL) για όλα τα σχετικά URLs της σελίδας.|

## Βέλτιστες Πρακτικές (Best Practices)

- Τοποθετείτε πάντα το `<head>` πριν από το `<body>`.
- Συμπεριλάβετε υποχρεωτικά το `<title>` σε κάθε έγγραφο HTML.
- Δηλώστε την κωδικοποίηση χαρακτήρων με:

```html
<meta charset="UTF-8">
```

- Χρησιμοποιήστε το viewport για σωστή εμφάνιση σε κινητές συσκευές:

```html
<meta
  name="viewport"
  content="width=device-width, initial-scale=1.0"
>
```

- Τοποθετείτε τα αρχεία CSS μέσα στο `<head>`.
- Προτιμήστε να φορτώνετε τα εξωτερικά αρχεία JavaScript με το attribute `defer`, ώστε να μην καθυστερεί η φόρτωση της σελίδας.

> **Σημείωση:** Το στοιχείο `<head>` **δεν εμφανίζεται** στην ιστοσελίδα. Περιέχει μόνο πληροφορίες και ρυθμίσεις που βοηθούν τον browser, τις μηχανές αναζήτησης και άλλες εφαρμογές να κατανοήσουν και να διαχειριστούν σωστά το έγγραφο HTML.
