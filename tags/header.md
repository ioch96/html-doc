# HTML `<header>` Tag

Το tag `<header>` χρησιμοποιείται για να ορίσει την **κεφαλίδα (header)** μιας ιστοσελίδας ή μιας θεματικής ενότητας.

Αντιπροσωπεύει ένα **σημασιολογικό στοιχείο (semantic element)** που περιέχει συνήθως το εισαγωγικό περιεχόμενο μιας σελίδας ή μιας ενότητας, καθώς και στοιχεία πλοήγησης.

Ένα στοιχείο `<header>` συνήθως περιέχει:

- Μία ή περισσότερες επικεφαλίδες (`<h1>` - `<h6>`).
- Το λογότυπο (logo) ή το εικονίδιο της ιστοσελίδας.
- Πληροφορίες για τον συγγραφέα ή τον οργανισμό.
- Στοιχεία πλοήγησης (`<nav>`).

## Παράδειγμα

```html
<!DOCTYPE html>
<html lang="el">
<head>
  <title>Το στοιχείο header</title>
</head>

<body>

  <header>
    <img src="logo.png" alt="Logo">

    <h1>My Website</h1>

    <nav>
      <a href="#">Αρχική</a>
      <a href="#">Άρθρα</a>
      <a href="#">Επικοινωνία</a>
    </nav>
  </header>

</body>
</html>
```

## Το `<header>` μέσα σε ένα `<article>`

Το `<header>` δεν χρησιμοποιείται μόνο στην κορυφή της σελίδας. Μπορεί επίσης να χρησιμοποιηθεί στην αρχή ενός άρθρου (`<article>`) ή μιας ενότητας (`<section>`).

Παράδειγμα:

```html
<article>

  <header>
    <h2>Τι είναι η HTML;</h2>

    <p>Δημοσιεύτηκε στις 15 Ιουλίου 2026</p>
  </header>

  <p>
    Η HTML είναι η βασική γλώσσα σήμανσης του Web...
  </p>

</article>
```

Εδώ το `<header>` αποτελεί την κεφαλίδα του άρθρου και όχι ολόκληρης της ιστοσελίδας.

## Μπορούν να υπάρχουν πολλά `<header>`

**Ναι.**

Ένα έγγραφο HTML μπορεί να περιέχει **περισσότερα από ένα** στοιχεία `<header>`.

Παράδειγμα:

```html
<header>
  <h1>My Blog</h1>
</header>

<main>

  <article>

    <header>
      <h2>Πρώτο άρθρο</h2>
    </header>

    <p>...</p>

  </article>

  <article>

    <header>
      <h2>Δεύτερο άρθρο</h2>
    </header>

    <p>...</p>

  </article>

</main>
```

Στο παραπάνω παράδειγμα υπάρχουν:

- Ένα `<header>` για ολόκληρη τη σελίδα.
- Ένα `<header>` για κάθε άρθρο.

## Πού **δεν** επιτρέπεται να χρησιμοποιηθεί

Ένα στοιχείο `<header>` **δεν μπορεί** να τοποθετηθεί μέσα σε:

- `<header>`
- `<footer>`
- `<address>`

## Βέλτιστες Πρακτικές (Best Practices)

- Χρησιμοποιείτε το `<header>` για το εισαγωγικό περιεχόμενο μιας σελίδας ή μιας ενότητας.
- Τοποθετείτε μέσα του στοιχεία όπως λογότυπο, τίτλους και μενού πλοήγησης.
- Μπορείτε να χρησιμοποιείτε περισσότερα από ένα `<header>` όταν υπάρχουν διαφορετικές θεματικές ενότητες ή άρθρα.
- Μην χρησιμοποιείτε το `<header>` μόνο για λόγους μορφοποίησης. Αν χρειάζεστε απλώς ένα γενικό container, χρησιμοποιήστε το `<div>`.

> **Σημείωση:** Το `<header>` είναι ένα **σημασιολογικό στοιχείο (semantic element)** που περιγράφει την κεφαλίδα μιας σελίδας ή μιας ενότητας. Χρησιμοποιείται ευρέως σε σύγχρονες ιστοσελίδες και εφαρμογές React, καθώς βελτιώνει την αναγνωσιμότητα του κώδικα, την προσβασιμότητα και το SEO.
